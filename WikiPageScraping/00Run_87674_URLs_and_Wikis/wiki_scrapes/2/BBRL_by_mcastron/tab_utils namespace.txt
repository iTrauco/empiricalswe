          <p>This namespace gathers generic tools, which can be used in many other contexts.</p>
<ul>
<li>
<strong>Chrono class</strong>, represents a chronometer to measure time.</li>
<li>
<strong>RandomGen class</strong>, used to draw random numbers.</li>
<li>
<strong>Thread class</strong>, represents a thread.</li>
<li>
<strong>algorithm namespace</strong>, gathers various algorithms <a href="http://homes.di.unimi.it/cesa-bianchi/Pubblicazioni/ml-02.pdf" rel="nofollow">UCB1</a>, <a href="https://dslpitt.org/uai/papers/07/p67-coquelin.pdf" rel="nofollow">UCT</a> and <a href="https://sequel.lille.inria.fr/Software/StoSOO" rel="nofollow">Sto-SOO</a>.</li>
</ul>
<h5></h5>
<ul>
<li>
<strong>statistics namespace</strong>, gathers a set of methods to compute statistical data.</li>
<li>
<strong>compression namespace</strong>, gathers a set of methods to compress data.</li>
<li>
<strong>search namespace</strong>, gathers a set of methods to find the minimal/maximal value of a <code>std</code> container.</li>
<li>
<strong>parsing namespace</strong>, gathers a set of methods to handle command-line options parsing.</li>
<li>
<strong>formula namespace</strong>, gathers a set of tools concerning the management of mathematical formulas.</li>
</ul>
<h5></h5>
<ul>
<li>
<strong>gnuplot namespace</strong>, gathers several classes and functions to create graphs using gnuplot.</li>
<li>
<strong>latex namespace</strong>, gathers several classes and functions to create latex tables.</li>
</ul>
