          <p><code>BBRL-zlib</code> is a program allowing you to compress/decompress the data files generated by <code>BBRL-DDS</code>.</p>
<h1>
<a aria-hidden="true" class="anchor" href="#classic-use" id="user-content-classic-use"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Classic use</h1>
<h2>
<a aria-hidden="true" class="anchor" href="#compress" id="user-content-compress"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Compress</h2>
<p>This is achieved through the following command-line options:</p>
<div class="highlight highlight-source-shell"><pre>./BBRL-zlib --compress --compression_level [...] <span class="pl-s"><span class="pl-pds">"</span>input.dat<span class="pl-pds">"</span></span></pre></div>
<p>This will create the output file <code>input.dat.zz</code>, which is a compressed version of <code>input.dat</code>.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#decompress" id="user-content-decompress"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Decompress</h2>
<p>This is achieved through the following command-line options:</p>
<div class="highlight highlight-source-shell"><pre>./BBRL-zlib --decompress <span class="pl-s"><span class="pl-pds">"</span>input.dat.zz<span class="pl-pds">"</span></span></pre></div>
<p>This will create the output file <code>input.dat</code>, which is a decompressed version of <code>input.dat.zz</code>.</p>
