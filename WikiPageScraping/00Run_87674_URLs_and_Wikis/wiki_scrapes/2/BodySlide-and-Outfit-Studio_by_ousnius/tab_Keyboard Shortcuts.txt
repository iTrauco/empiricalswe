<ul>
<li>Hold <strong>CTRL</strong> while clicking the Build button to build to the current working directory.</li>
<li>Hold <strong>CTRL</strong> while clicking the Batch Build button to select a target directory.</li>
<li>Hold <strong>ALT</strong> while clicking the Build button to delete the target files.</li>

<h3>
<a aria-hidden="true" class="anchor" href="#brushing" id="user-content-brushing"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Brushing</h3>
<ul>
<li>Press <strong>0-6</strong> to select a different brush.</li>
<li>Hold <strong>S</strong> and use the mouse wheel to quickly adjust the brush size.</li>
<li>Hold <strong>CTRL</strong> to paint the mask with any brush selected.</li>
<li>Hold <strong>CTRL+ALT</strong> to erase the mask with any brush selected.</li>
<li>Press <strong>CTRL+A</strong> to clear the mask.</li>
<li>Press <strong>CTRL+I</strong> to invert the mask.</li>
<li>Press <strong>CTRL+Z</strong> to undo the last brush stroke.</li>
<li>Press <strong>CTRL+Y</strong> to redo the last brush stroke.</li>
</ul>

<ul>
<li>Press <strong>X</strong> to toggle the X-Mirror.</li>
<li>Press <strong>C</strong> to toggle the Edit Connected Only mode.</li>
<li>Press <strong>B</strong> to toggle the Global Brush Collision for multiple shapes.</li>
<li>Press <strong>Q</strong> to toggle the vertex selection mode.</li>
<li>Press <strong>F</strong> to toggle the transform tool.</li>
<li>Press <strong>V</strong> to show a dialog for moving only the single vertex your mouse currently is on.</li>
</ul>

<li>Press <strong>G</strong> to toggle the ghost mode.</li>
