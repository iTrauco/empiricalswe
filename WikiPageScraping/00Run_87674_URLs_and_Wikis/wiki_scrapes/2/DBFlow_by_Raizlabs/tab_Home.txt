<ol>
<li>Library now is on jCenter()/bintray!</li>
<li>Full Enum Support. Note: they must be standard columns. We do not support foreign key or primary key for enums.</li>
<li>Can now define inherited properties to use as columns. Note: They can only be normal, accessible to the subclass columns for now. Just
define <code>@Table(inheritedColumns = {@InheritedColumn(column = @Column, fieldName = "fieldName")}</code>.</li>
<li>Bug Fixes, readme enhancements, Logging Improvements, and improved code commenting</li>
<li>Function support for SQLite methods in a condition such that <code>date(myColumn1)=1433872730</code> can be written
as <code>Condition.columnsWithFunction("date", "myColumn1").eq(1433872730)</code>
</li>
<li>Fixed an issue where <code>Condition</code> instead of <code>SQLCondition</code> were leftover as a param in a few methods.</li>
</ol>
