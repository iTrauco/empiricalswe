<h1>
<a aria-hidden="true" class="anchor" href="#success-criteria" id="user-content-success-criteria"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Success Criteria</h1>
<p>To determine if the project is successful, the final product must do a few things. If all of these criteria are met, we can be sure that we have met all of the requirements for this project.</p>
<ol>
<li>The microcontroller must be able to read CAN messages from both of the two CAN buses the electric race car has.</li>
<li>The microcontroller must be able to send data read from the CAN buses to the LCD screen.</li>
<li>The LCD screen must be able to interpret the data being received from the microcontroller.</li>
<li>The LCD screen displays the data in a simple and readable way for the driver.</li>
<li>The LCD screen is readable in direct/harsh sunlight.</li>
</ol>
<h1>
<a aria-hidden="true" class="anchor" href="#potential-roadblocks" id="user-content-potential-roadblocks"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Potential Roadblocks</h1>
<p>The largest potential roadblock will be the LCD screen. The 4D systems screen that we have chosen is not transreflective, meaning that it does not do well in harsh sunlight from the factory. To address this issue, we plan to add a polarizing shield and a shroud to enable the driver to see the screen in harsh sunlight. If this idea does not work as intended, we will need to source another screen that is transreflective (and more expensive). Whether or not we need this second screen will be determined after the prototype has been created.</p>
<p>Another potential roadblock will be waterproofing the enclosure. The enclosure must be able to withstand a two minute water test. Multiple ideas have been proposed on how to go about this, so in the end we will come out with an enclosure that will pass all Formula SAE water tests.</p>
<p>Beyond these two issues, everything else in the project should go off without a hitch. We, of course, are prepared to handle unexpected issues during the course of the project.</p>
