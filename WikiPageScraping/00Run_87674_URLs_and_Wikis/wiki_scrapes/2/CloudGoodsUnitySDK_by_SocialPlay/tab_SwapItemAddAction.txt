          <p>This component is added to a container to give the container a basic add to container action.</p>
<p><strong>public int swapIndex -</strong> The list index of the item that will be swapped</p>
<p><strong>public int swapItemLimit -</strong> The limit of the amount of unique items before an item will be swapped</p>
<p><strong>public void AddItem -</strong> The implementation of the interfaced method AddItem. This implementation will add an item if there is less items than the swapItemLimit. If there are more and there is an attempt to add, it will swap with the given indexed item instead.</p>
