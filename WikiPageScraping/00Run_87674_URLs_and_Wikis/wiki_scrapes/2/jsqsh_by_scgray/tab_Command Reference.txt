a number of commands come with built-in aliases for convenience
(see the <strong>Alias(s)</strong> column).  Most jsqsh command take additional
command line arguments, for details on how command line arguments
work within jsqsh see <a class="internal present" href="/scgray/jsqsh/wiki/options">options</a>.</p>
