          <p>Since the Closure Compiler is able to read your JavaScript code it also can perform refactoring on it. Therefore Google's RefasterJS takes a before and after template of your code. Simply but these template files in you project in a folder called "refasterjs".</p>
<p>Checkout the todo example that includes a simple refaster demo
<a href="https://github.com/silksmith/silksmith-todo-sample">https://github.com/silksmith/silksmith-todo-sample</a></p>
