<blockquote>
<p>The goal is to implement <em>module_sound</em> in order to further refactor and modularize Inexor's code.</p>
</blockquote>
<p>Until now Inexor is using Cube2's old sound system which uses SDL_Mixer. It has no class design and is not documented at all which makes maintaining it quite unpleasant. We will also implement new features for inexor's sound system.
This code is mostly separated from other code parts so the refactoring progress shouldn't be too complicated.</p>
<p>This task is divided into 4 steps:</p>
