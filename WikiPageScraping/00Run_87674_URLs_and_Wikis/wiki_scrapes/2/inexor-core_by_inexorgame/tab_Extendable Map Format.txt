<a aria-hidden="true" class="anchor" href="#maps-can-be-loaded-by-older-and-newer-clients" id="user-content-maps-can-be-loaded-by-older-and-newer-clients"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Maps can be loaded by older and newer clients</h4>
<ul>
<li>Older servers and clients profit from maps build by mappers with a newer client</li>
<li>Features saved in the maps are ignored if an older client don't know them</li>
</ul>
<h4>
<a aria-hidden="true" class="anchor" href="#store-more-complex-data-structures" id="user-content-store-more-complex-data-structures"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Store more complex data structures</h4>
<ul>
<li>The new entity (+particle) system can't be saved in such a static way the old map format is (5 integers for each attribute)</li>
<li>References to internal data (Lightmap references an octree node)</li>
<li>References to external data (URLs to other content)</li>
<li>Serialization / Deserialization</li>
<li>Compression</li>
</ul>

<ul>
<li>Plugins, extensions and mods may have to store additional data in a map (for example for special game modes)</li>
<li>External tools, websites and the API can read the data</li>
</ul>
