<p>###Installation</p>
<p>Latest available file:</p>
<ul>
<li>uSkyblock v2.6.12 for MC 1.8, 1.9 or 1.10</li>
</ul>
<p>Required dependencies are:</p>
<ul>
<li>WorldEdit v6.xx (get it <a href="http://dev.bukkit.org/bukkit-plugins/worldedit/files/63-world-edit-6-1-2-up-to-mc-1-9-4/" rel="nofollow">here</a>, or click <a href="http://builds.enginehub.org/job/worldedit?branch=master" rel="nofollow">here</a> for Dev builds)</li>
<li>WorldGuard v6.xx (get it <a href="http://dev.bukkit.org/bukkit-plugins/worldguard/files/46-world-guard-6-1/" rel="nofollow">here</a>, or click <a href="http://builds.enginehub.org/job/worldguard?branch=master" rel="nofollow">here</a> for Dev builds)</li>
<li>Vault v1.5.x (get it <a href="http://dev.bukkit.org/bukkit-plugins/vault/files/52-vault-1-5-6/" rel="nofollow">here</a>)</li>
<li>Multiverse-Core (versions above v2.5) (get it <a href="http://dev.bukkit.org/bukkit-plugins/multiverse-core/files/27-2-5-b691-beta/" rel="nofollow">here</a>, or click <a href="http://ci.onarandombox.com/job/Multiverse-Core/" rel="nofollow">here</a> for Dev builds)\</li>
<li>Multiverse-Portals (versions above v2.5) (get it <a href="http://dev.bukkit.org/bukkit-plugins/multiverse-portals/files/14-2-6-b725-beta/" rel="nofollow">here</a>, or click <a href="http://ci.onarandombox.com/job/Multiverse-Portals/" rel="nofollow">here</a> for Dev builds)</li>
<li>Multiverse-NetherPortals (versions above v2.5) (get it <a href="http://dev.bukkit.org/bukkit-plugins/multiverse-netherportals/files/11-2-5-b675-beta/" rel="nofollow">here</a>, or click <a href="http://ci.onarandombox.com/job/Multiverse-NetherPortals/" rel="nofollow">here</a> for Dev builds)</li>
</ul>
<p>Recommended soft dependencies are:</p>
<ul>
<li>Chest Commands GUI (get it <a href="http://dev.bukkit.org/bukkit-plugins/chest-commands/files/52-chest-commands-v3-1-4/" rel="nofollow">here</a>)</li>
<li>Fawe (get it <a href="https://www.spigotmc.org/resources/free-fast-async-worldedit-voxelsniper-%CE%B2eta.13932/" rel="nofollow">here</a>)</li>
<li>AWE (get it <a href="https://www.spigotmc.org/resources/asyncworldedit.327/" rel="nofollow">here</a> free version or click <a href="https://www.spigotmc.org/resources/asyncworldedit-premium.9661/" rel="nofollow">here</a> for the premium version).</li>
</ul>
<p>Stop your server and place the uskyblock.jar and all depended plugin jars in your plugins folder (<em>you can use a plugin reloader like Plugman but we don't recommend this</em>).<br/>
Restart your server to let uSkyblock create your skyworld and all necessary files in the plugin folder (this include files in WorldGuard folder).<br/>
Alter the config(s) to your own liking (use <strong>/usb reload</strong> or restart the server) or leave them at default, uskyblock works strait out the box.</p>
<p><strong>The support for 1.7 servers and all clones of it has been discontinued.</strong></p>
