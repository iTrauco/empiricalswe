          <p>It seems many people here are very familiar with MATLAB, Octave and LaTeX but less so with git. In this page, we shall give a short overview of other sources that cover git or practical tools that can help you to ease your work with git.
The materials are subdivided into different categories where the most relevant items are mentioned first.</p>
<h1>
<a aria-hidden="true" class="anchor" href="#written-material" id="user-content-written-material"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Written material</h1>
<ul>
<li>
<a href="http://git-scm.com/doc" rel="nofollow">The Official documentation</a> contains a lot of interesting materials and a lot of links like the ones below.</li>
<li>
<a href="http://ericsink.com/vcbe/" rel="nofollow">Version Control by Example (Eric Sink)</a> is ideal if you're new to version control. It covers more than git, but starts from the very basics.</li>
<li>
<a href="http://git-scm.com/book/en/v2" rel="nofollow">Pro Git by Scott Chacon</a> is the git reference: it's a great read and it's free! Never go anywhere without it.</li>
<li>
<a href="https://help.github.com/">GitHub help</a> and <a href="https://guides.github.com/">GitHub guides</a> cover git and how you should interact with it on GitHub in specific.</li>
</ul>

<p>While it's advised to read a tutorial at your own pace, sometimes it is just more convenient to watch a video that spoonfeeds you git.</p>
<ul>
<li>
<a href="http://youtu.be/1ffBJ4sVUb40" rel="nofollow">Git for ages 4 and up</a> to learn to see how you can work with branches.</li>
<li>
<a href="https://www.youtube.com/watch?v=ZDR433b0HJY" rel="nofollow">Introduction to git with Scott Chacon from GitHub</a> covers the basics, but also delves more into how everything is stored.</li>
</ul>

<ul>
<li>Since MATLAB R2014b, there is built-in git support in MATLAB.</li>
<li>There is also quite some git tools available on the <a href="">File Exchange</a>:
<ul>
<li>Just a thin wrapper for the git command line: e.g. <a href="http://www.mathworks.com/matlabcentral/fileexchange/29154-a-thin-matlab-wrapper-for-the-git-source-control-system" rel="nofollow">this one</a> or <a href="http://www.mathworks.com/matlabcentral/fileexchange/38600-git-matlab" rel="nofollow">another one</a>
</li>
<li>
<a href="http://www.mathworks.com/matlabcentral/fileexchange/41348-jgit4matlab" rel="nofollow">JGit for MATLAB</a> is a Java-based (but entirely compatible) implementation of git</li>
</ul>
</li>
</ul>
<h2>
<a aria-hidden="true" class="anchor" href="#gui-tools" id="user-content-gui-tools"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>GUI Tools</h2>
<p>Often it's easier to use a GUI to access git than fiddle with the command line. These don't always provide a full interface to the possibilities, but may make you more comfortable using git.</p>
<ul>
<li>
<a href="https://www.atlassian.com/software/sourcetree/overview" rel="nofollow">SourceTree</a> is a very good OS X &amp; Windows client.</li>
<li>
<a href="http://www.syntevo.com/smartgit/" rel="nofollow">SmartGitHg</a> is a great crossplatform tool to access git, but unfortunately not free.</li>
<li>On Windows, you can use <a href="https://code.google.com/p/tortoisegit/" rel="nofollow">TortoiseGit</a> to integrate git into Windows explorer.</li>
</ul>
