<a aria-hidden="true" class="anchor" href="#introduction" id="user-content-introduction"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>INTRODUCTION</h1>
<p>This document is going to teach you how to make My Block icons and My Block Parameter icons. This would be useful because the My Block icons would be easier to recognize in the mission programs. We are the first ones to document how to use this so technically were the first to find this out. We broke this document down into two parts, My Block Icons, and Parameter Icons. You can only use this with the EV3 software, and the programs only display properly when and where you install your custom icons. Before we begin, make a copy of <strong><em>\LEGO MINDSTORMS Edu EV3\Resources\MyBlocks\images</em></strong> directory.</p>
<h1>
<a aria-hidden="true" class="anchor" href="#parameter-icons" id="user-content-parameter-icons"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>PARAMETER ICONS</h1>
<p>First you are going to open up a picture editing software so you can create an icon, we used Gimp and we recommend using it. When you’re creating the icon keep in mind it needs to be 22x22 pixels. You can use pictures from the MyBlocks/images directory or you can make you own icon from scratch. Then you need to export the image as a .png and save it in the My Block image directory. The name has to be in this format: the word  “Identification”, Underscore, a 3 digit number, Underscore, Icon Name (no spaces, upper camel cased and underscores OK) , and “.png” Example: Identification_100_TestBlock.png The 3 digit number has to be an unused number, 100-199 are open. Before your icon shows up you need to restart your EV3 Software. Then reopen it and your icon should be there!</p>

<p>Again, with your picture editing software you’re going to open it up and make another icon, but this time make it 34x34 pixels. After you’re done making the icon, the steps get a bit different than the parameter icons. Export it again as a .png but you need to name it “PolyGroup”, Underscore, 3 digit number, underscore, icon name (No spaces, upper camel case, and NO underscores) ,underscore, ”Diagram”, and “.png”.   Example:  PolyGroup_019_TestBlock_Diagram.png.  The 3 digit number has to be an unused number, 002-019 are open. Then shrink the icon in the editing software to 25x25 and export it again with the same name but replace the word diagram with “Pallet”. Now you need to make one more the same size but add a highlite for a mouse over. Add an underscore after the pallet and add “MouseOver” When exporting. Again, restart your EV3 software and there you will have it.
The Diagram icon is used when you view the icon in the My Block Creation Menu.
The Pallet icon is used when you pick a My Block from the My Block pallet.
The MouseOver icon is used when you scroll over the block in the My Block Pallet.</p>
