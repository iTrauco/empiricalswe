          <p>You only need to run the script once a day after you've completed all your activities.</p>
<p><strong>NOTE:</strong> you shoul always use the format DD-MMM-YYYY e.g.: 18-Nov-2014</p>
<pre><code>cd &lt;INSTALL_DIR&gt;  
python Toggl_to_Gcal.py &lt;DAY&gt;
python Toggl_to_Gcal.py 18-Nov-2014
</code></pre>
