<ul>
<li><a href="http://wiki.c2b2.columbia.edu/dream/index.php/D5c4" rel="nofollow">DREAM5 network inference challenge description</a></li>
</ul>
<p><strong>Common formatting errors of the expression data</strong></p>
<ul>
<li>Genes are given in the columns, not rows (see <a href="http://www.broadinstitute.org/%7Emarbach/net1_input_data.zip" rel="nofollow">example data</a>).</li>
<li>Some modules do now allow numbers as gene IDs: <code>G1</code> is ok, <code>1</code> is not.</li>
<li>When using Windows, make sure that the encoding of the text file is unix compatible (sometimes there are problems with line breaks).</li>
</ul>
