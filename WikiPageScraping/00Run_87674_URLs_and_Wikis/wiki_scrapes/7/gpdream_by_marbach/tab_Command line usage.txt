          <p>GP-DREAM modules are simply command-line applications. The GenePattern platform merely provides an easy-to-use interface to launch them. Thus, modules can also be run independently as command-line tools. For users who are comfortable with working from the command line, <strong>we recommend to install and run modules locally</strong> to avoid [wait time and resource limits](Runtime &amp; memory limits) on the Broad cluster.</p>

<p>How to run modules from the command-line is described in the <a href="Modules">Module documentation</a> pages. Depending on your system and the module, this can be straightforward or involve installation of additional software and libraries.</p>
