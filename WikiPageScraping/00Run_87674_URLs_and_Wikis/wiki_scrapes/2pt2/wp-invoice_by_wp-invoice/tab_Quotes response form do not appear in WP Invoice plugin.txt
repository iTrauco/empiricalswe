          <p>In most of cases issue is in that your Page Template that you use for Invoices/Quotes does not support comments.</p>
<p>Read this to understand how to add comments support for page template:<a href="http://codex.wordpress.org/Function_Reference/comment_form" rel="nofollow">http://codex.wordpress.org/Function_Reference/comment_form</a></p>
<p>﻿The other frequent question is ‘How to display the list of existing responses?’.</p>
<p>So, to get responses on quote page use this standard wordpress function:</p>
<p><a href="http://codex.wordpress.org/Function_Reference/wp_list_comments" rel="nofollow">http://codex.wordpress.org/Function_Reference/wp_list_comments</a></p>
<p>Actually, these function usually used for regular wordpress post types. You can find code examples in default themes ‘Twenty Ten’ and ‘Twenty Eleven’.</p>
