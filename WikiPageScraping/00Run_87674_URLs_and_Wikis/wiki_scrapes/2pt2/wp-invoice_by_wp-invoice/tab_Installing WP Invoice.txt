          <p>WP-Invoice is a free plugin for WordPress and is available for download from the official <a href="https://wordpress.org/plugins/wp-invoice/" rel="nofollow">WordPress Plugin Directory</a>. WP-Invoice supports both WordPress manual and automatic plugin <a href="https://codex.wordpress.org/Managing_Plugins#Installing_Plugins" rel="nofollow">installation methods</a>.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#automatic-installation" id="user-content-automatic-installation"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Automatic Installation</h2>
<p>Automatic installation is the easiest and most popular method for installing WP-Invoice.</p>
<ol>
<li>From your WordPress dashboard, navigate to <em>Plugins</em> &gt; <em>Add New</em>.</li>
<li>Under the <em>Search</em> subtitle, search for <em>WP-Invoice</em>.</li>
<li>From the search results, click the <em>Install Now</em> link for <em>WP-Invoice - Web Invoice and Billing</em> authored by <em>Usability Dynamics, Inc.</em>
</li>
<li>Click <em>OK</em> on the confirmation dialog to confirm the installation action.</li>
</ol>
<h2>
<a aria-hidden="true" class="anchor" href="#manual-installation" id="user-content-manual-installation"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Manual Installation</h2>
<p>Manual installation is available for those whose server does not permit automatic installation or for any other reason.</p>
<ol>
<li>
<a href="http://wordpress.org/plugins/wp-invoice/" rel="nofollow">Download WP-Invoice</a> from the official WordPress Plugin Directory.</li>
<li>Extract the zip file onto your desktop.</li>
<li>With your FTP program, upload the extracted <em>WP-Invoice</em> folder to the <em>wp-content/plugins</em> folder in your WordPress directory online.</li>
<li>From your WordPress dashboard, navigate to the <em>Plugins</em> page.</li>
<li>Click the Activate link for <em>WP-Invoice - Web Invoice and Billing</em> authored by <em>Usability Dynamics, Inc.</em>
</li>
</ol>
