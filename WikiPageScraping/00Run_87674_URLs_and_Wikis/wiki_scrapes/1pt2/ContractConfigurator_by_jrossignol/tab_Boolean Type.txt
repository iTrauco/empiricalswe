          <p>The boolean data type is referred to as a <code>bool</code>.  It has two values (true/false) which are case insensitive when referring to them in expressions.</p>
<p>There are a number of special operators that use booleans - for more details see the main <a class="internal present" href="/jrossignol/ContractConfigurator/wiki/Expressions">Expressions</a> page.</p>
<p>There are no functions/methods that operator on boolean values.</p>
