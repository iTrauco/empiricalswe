          <p>Anonymous functions can be created using the <code>lambda</code> keyword. Anonymous functions have access to all local variables in their parent function. The syntax for the lambda keyword is</p>
<pre><code>lambda (&lt;parameters&gt;) &lt;statement&gt;
</code></pre>
<p><strong>Example</strong></p>
<pre><code>addTwo = lambda (x, y) return x + y;
print (addTwo (2, 3));
</code></pre>
