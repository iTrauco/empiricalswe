          <p>Enumerations in Iodine are defined using the <code>'enum</code> keyword. The syntax for declaring an <code>enum</code> is</p>
<pre><code>enum MyEnum {
    VALUE_1,
    VALUE_2
}
</code></pre>
<p>Values may also be assigned a constant numeric value. For example</p>
<pre><code>enum MyEnum {
    VALUE_1 = 0,
    VALUE_2 = 1
}
</code></pre>
