          <p><a href="Home">Home</a> <br/>
<b>Features:</b><br/>
  Picture based security System.<br/>
  Check out child with a bar code scanner.<br/>
  Junior and High School leaders can check in students with a smart phone. (no tag printed)<br/>
  Being web based, Windows, MacOS and Linux computers can access the system.<br/>
  Being web based, Mobile phones running Windows, MacOS and Android can access the system.<br/>
  Visitor tags for families that are "church hopping".<br/>
  Open Source means anything and everything can be modified.<br/></p>
<p><b>Advanced admin options:</b><br/>
  To see child check in/out times for any date in the system.<br/>
  Manage Volunteers/Families/Children/Adults.<br/>
  Manage families and who can pickup kids. Valid parents that can pick up kids can be changed on the fly at the check in stations. (username and password required to change family information)<br/>
  Most people have a mobile phone with unlimited text messages. So there is an option to send text messages to parents of children in trouble via a volunteers smart phone. (no texting charges for the church)<br/>
  Add complete families at one time<br/>
  Export parent email addresses of age groups in Windows format or MacOS format. i.e. Staff only wants to email one year old parents, the system can export only the parents email of the one year old class.<br/></p>
<p><b>Requirements:</b><br/>
  ANY computer with a web browser and a Dymo printer can check in children/volunteers and print tags.<br/>
  Sunnybrook Christian Church (<a href="http://www.sunnybrookcc.org/" rel="nofollow">http://www.sunnybrookcc.org/</a>) uses 3 Windows Vista Laptops and 3 DYMO LabelWriter 450 Turbo Thermal Label Printers.<br/>
  An Information Technology person to help setup hardware and web server.<br/>
  Server Computer. (most churches today have some kind of server, this can piggy back on it OR it can be a old computer dedicated to check in)<br/></p>
