<p>Atomic Editor releases are scheduled as rolling <a href="https://github.com/AtomicGameEngine/AtomicGameEngine/milestones">milestones.</a>  If you are interested in contributing code, art, documentation, examples, or anything else, awesome üëç , please see: <a href="How-to-help-make-Atomic-better">How to help make Atomic better.</a></p>
<p><strong>C# Support <sup>Optional</sup></strong></p>
<p>The Atomic Editor integrates with <a href="https://www.visualstudio.com/vs/community/" rel="nofollow">Visual Studio</a>, <a href="https://www.visualstudio.com/vs/visual-studio-mac/" rel="nofollow">Visual Studio for Mac</a>, and <a href="Detailed-instructions-for-building-on-Linux">MonoDevelop</a> to provide a first class editing, debugging, and deployment experience on desktop and mobile platforms.  Install the IDE for your platform to enable C# scripting support.</p>

<img align="right" src="images/repo/setting-up-atomic/atomicregister.gif"/>
<p>On the initial run, the editor will confirm licensing terms and provide registration for a new Atomic Community ID or to enter an existing one.</p>
<p>During registration you can optionally sign up for updates and anonymous editor usage metrics. Registration is a spam free zone with no information being shared and only notification of significant updates.</p>
