          <p>Documented here is the current state of play for what coverage tools are supported, which are in progress, and a wish-list of tools to support in the future.</p>

<ul>
<li><a href="https://github.com/sawilde/opencover">OpenCover</a></li>
<li><a href="http://www.mono-project.com/docs/debug+profile/profile/code-coverage/" rel="nofollow">Mono Code Coverage (monocov)</a></li>
<li>
<a href="http://msdn.microsoft.com/en-us/library/dd299398%28v=vs.90%29.aspx" rel="nofollow">Visual Studio's Dynamic Coverage</a> (based on <a href="https://reportgenerator.codeplex.com/wikipage?title=Visual%20Studio%20Coverage%20Tools" rel="nofollow">ReportGenerator's support</a>,  using vstest.console.exe and CodeCoverage.exe)</li>
<li>
<a href="http://msdn.microsoft.com/en-us/library/dd299398%28v=vs.90%29.aspx" rel="nofollow">Visual Studio Coverage Export</a> (also covered by <a href="https://reportgenerator.codeplex.com/wikipage?title=Visual%20Studio%20Coverage%20Tools" rel="nofollow">ReportGenerator's support</a>, not sure if it can be command-line driven)</li>
<li>
<a href="https://github.com/mmanela/chutzpah">chutzpah</a> (A JavaScript Test Runner coverage results)</li>
<li>LCOV format</li>
</ul>
