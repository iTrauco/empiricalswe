<p>Coveralls uploader for .Net Code coverage of your C# source code. Should work with any code files that get reported with the supported coverage tools, but the primary focus is CSharp.</p>
<p>The aim of this tool is to be the go-to .Net Coverage publisher to <a href="https://coveralls.io/" rel="nofollow">coveralls.io</a> for all code coverage tools that work with .Net code, on windows and linux/mac (with mono).</p>

<p>Sample applications using Coveralls.net to publish their results can be found in the <a href="https://github.com/csmacnz/Coveralls.net-Samples">csmacnz/Coveralls.net-Samples</a> Project.</p>

<p>The package is available on nuget, which includes the .Net executable that runs on .net on windows and mono on linux.  You can view the <a href="http://nuget.org/packages/coveralls.net/" rel="nofollow">nuget package page</a> for more details.</p>
