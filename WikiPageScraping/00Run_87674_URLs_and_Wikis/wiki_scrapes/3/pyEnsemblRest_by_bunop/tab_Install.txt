<p>Clone the pyEnsemblRest then install package from source (use sudo or became <code>root</code>
to install system wide):</p>
<div class="highlight highlight-source-shell"><pre>$ git clone https://github.com/pyOpenSci/pyEnsemblRest.git
$ <span class="pl-c1">cd</span> pyEnsemblRest
$ python setup.py install</pre></div>
