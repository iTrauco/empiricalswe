          <p>todo: installing providers</p>
<p>$ putenv plugin install LOCATION</p>
<p>LOCATION = URL of git repo containing the provider plugin code. The tool will confirm that you do not already have the provider installed.</p>
<p>$ putenv plug list</p>
<p>List all of the installed provider plugins and version</p>
<p>$ putenv update [PLUGIN]</p>
<p>Command will check to see of the provider plugin repo (master) contains a different version than you have installed and will update to new version if available. If you execute command with no provider specified the tool will update all of the installed provider plugins.</p>
<p>$ putenv plugin remove PLUGIN</p>
<p>Uninstalls the specified plugin.</p>
