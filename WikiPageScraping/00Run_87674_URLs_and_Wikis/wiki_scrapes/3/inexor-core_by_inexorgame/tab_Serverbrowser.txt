          <p>Players want to join a game with others.
There are two solutions in the wild to achieve that:</p>
<ul>
<li>match making</li>
<li>a Serverbrowser</li>
</ul>
<p>while the two only differ in the regard that match making selects the server for you automatically.
The Serverbrowser on the other hand is a user interface so people can choose their server on their own.</p>
<p>What you need for both is, though:
The list of all available servers.</p>
<p>The blueprint we implement is</p>
<p><a class="internal present" href="/inexorgame/inexor-core/wiki/Decentralized-server-list">Decentralized server list</a></p>
<p>There existed different older proposals to make the masterserver more robust to DDoS-Attacks, but they we're ultimately discarded. We think we have a pretty good solution found now!</p>
