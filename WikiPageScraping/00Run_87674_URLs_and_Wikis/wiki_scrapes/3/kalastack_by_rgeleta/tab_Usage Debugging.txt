          <p>Debugging is best done with an IDE such as netbeans or eclipse or with a text editor
such as SublimeText.</p>
<p>The xdebug setup should already be done on the Kalastack end so you should
only need to set up your IDE. Here is some useful documentation for
<a href="http://brianfisher.name/content/drupal-development-environment-os-x-mamp-pro-eclipse-xdebug-and-drush" rel="nofollow">eclipse</a>,
<a href="http://wiki.netbeans.org/HowToConfigureXDebug" rel="nofollow">netbeans</a> and
<a href="https://github.com/martomo/SublimeTextXdebug">SublimeText</a>.</p>
