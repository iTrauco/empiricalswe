          <p>To use this download the plugin and start Blender</p>
<p>File -&gt; User Preferences</p>
<p>Find the "Addons" tab and at the bottom click the "Install from File..." button. Navigate to the python script here, select the file and click "Install from File..." in the upper right corner. This should show the plugin:</p>
<p>Import and visualize gcode files generated by Slic3r (.gcode)</p>
<p>Click the checkbox to enable the plugin and close the user preferences window.</p>
<p>Now the "File-&gt;Import" menu should contain a new entry "Slic3r GCode (.gcode)" which you can use to import your gcode file.</p>
