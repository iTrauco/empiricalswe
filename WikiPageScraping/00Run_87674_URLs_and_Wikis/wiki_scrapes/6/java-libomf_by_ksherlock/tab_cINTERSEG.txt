          <p>$F6 <strong>cINTERSEG</strong> â€“ This record is the compressed version of the <a class="internal present" href="/ksherlock/java-libomf/wiki/INTERSEG">INTERSEG</a> record. It is identical to the <a class="internal present" href="/ksherlock/java-libomf/wiki/INTERSEG">INTERSEG</a> record, except that the offsets are 2 bytes long rather than 4 bytes, the segment number is 1 byte rather than 2 bytes, and it does not include the 2-byte file number. The <a class="internal present" href="/ksherlock/java-libomf/wiki/cINTERSEG">cINTERSEG</a> record can be used only if both offsets are less than $FFFF (65535), the segment number is less than 256, and the file number associated with the reference is 1. References to segments in run-time-library files must use <a class="internal present" href="/ksherlock/java-libomf/wiki/INTERSEG">INTERSEG</a> records rather than <a class="internal present" href="/ksherlock/java-libomf/wiki/cINTERSEG">cINTERSEG</a> records.</p>
