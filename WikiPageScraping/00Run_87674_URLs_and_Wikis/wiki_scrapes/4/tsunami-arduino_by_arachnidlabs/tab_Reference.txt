<h4>
<a aria-hidden="true" class="anchor" href="#description" id="user-content-description"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Puts the signal generator on the Tsunami to sleep, if true, or wakes it up, if false.</p>
<p>When the signal generator is asleep, it stops generating a signal, but otherwise continues operating as normal; when woken up again it will continue as if nothing happened.</p>

<h4>
<a aria-hidden="true" class="anchor" href="#description-1" id="user-content-description-1"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Puts the signal generator on the Tsunami into or out of reset.</p>
<p>When the signal generator is in reset, it still accepts commands such as <a href="#setFrequency">setFrequency</a>, but the output is held at mid-scale (the value set by <a href="#setOffset">setOffset</a>). As soon as reset is disabled, the Tsunami will start generating a signal again.</p>

<h4>
<a aria-hidden="true" class="anchor" href="#description-2" id="user-content-description-2"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Sets the amplitude (voltage range) of the Tsunami's output signal, in millivolts. The amplitude is the difference between the lowest voltage the Tsunami outputs and the highest value it outputs, so an amplitude of 3000 millivolts would mean that it swings from -1.5 volts to 1.5 volts, or from 0 volts to 3 volts, depending on the value passed to <a href="#setoffsetnum">setOffset</a>.</p>

<a aria-hidden="true" class="anchor" href="#setfrequency" id="user-content-setfrequency"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>setFrequency()</h3>
<h4>
<a aria-hidden="true" class="anchor" href="#description-3" id="user-content-description-3"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Sets the output frequency for the signal generator, in hertz.</p>
<p>When only one parameter is provided, this function both sets the output frequency and seamlessly switches to generating it.</p>
<p>When two parameters are provided, this function sets one of the two frequency registers to the desired frequency, but does not change which frequency register is currently selected; this must be done with <a href="#selectFrequency">selectFrequency</a>. Note that changing the frequency of the currently selected register when the signal generator is not sleeping or in reset may lead to glitches in the output signal.</p>

<p>Sets the offset voltage for the signal generator, in millivolts.</p>
<p>The offset voltage defines the voltage that the output signal is centred around. With an offset voltage of 0 (the default) and an amplitude of 1 volt, for instance, the output signal will go from -0.5 volts to 0.5 volts. With an offset of 0.5 volts and an amplitude of 1 volt, the output signal will go from 0 volts to 1 volt.</p>
<p>Offset may be any value from approximately -3.3 volts to 3.3 volts, but if the offset plus the amplitude exceeds approximately + or - 4 volts, the output may be clipped.</p>

<p>Measures the frequency of the signal on the Tsunami's input connector, returning a floating point number in hertz. Presently, this works for signals between approximately 32Hz and 8MHz, and has very high accuracy.</p>
<p>If the Tsunami is unable to accurately measure the frequency (for instance, because it has just changed dramatically, or because there is no input signal), it will return NAN.</p>

<h4>
<a aria-hidden="true" class="anchor" href="#description-8" id="user-content-description-8"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Measures the phase of the signal on the Tsunami's input connector, relative to the signal on the Tsunami's output connector. Returns a floating point number between 0 and 1.</p>
<p>This is useful when you want to measure the delay caused by a circuit you are testing by connecting both the Tsunami's output and input to it. If the input signal is not in some way generated or caused by the Tsunami's output, this function will return nonsensical and inconsistent results.</p>
<p>A phase of 0 indicates the signals are exactly in sync, while a phase of 1 indicates they are 180 degrees out of sync. Phase differences greater than 180 degrees will be 'mirrored' - so a phase difference of 181 degrees will return the same value as a phase difference of 179 degrees.</p>

<h4>
<a aria-hidden="true" class="anchor" href="#description-9" id="user-content-description-9"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Description</h4>
<p>Configures the auxiliary connector on the Tsunami to output a square wave. The square wave is high when the output signal on the main connector is above its midpoint, and low when the output signal is below its midpoint. This is a convenient way to generate a 5V square wave from the Tsunami.</p>
<p>Note that when generating high frequency signals above about 1MHz, the square wave may have significant jitter - that is, it may vary in period from cycle to cycle - unless the frequency you are generating is an exact fraction of 16MHz.</p>
