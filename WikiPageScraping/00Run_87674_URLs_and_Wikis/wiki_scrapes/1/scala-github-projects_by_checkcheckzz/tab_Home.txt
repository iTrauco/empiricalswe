          <p>##Add New Projects</p>
<p>The first line is <code>1.</code> + project name.<br/>
The second line is the description of the project.<br/>
The third line is <code>Project Source</code>: + url.<br/>
The three lines above are necessary. Followed by optional lines as follows:</p>
<p>Project Homepage: <code>Project Homepage</code>: + url.<br/>
or<br/>
Project Documentation:  <code>Project Documentation</code>: + url.<br/>
or<br/>
Online Demo Address: <code>Online Demo Address</code> + url.
<br/>
<br/>
Note: If a project does not have homepage, documentation and online demo, it should at least have a detailed description including some simple demos on the source website.</p>
