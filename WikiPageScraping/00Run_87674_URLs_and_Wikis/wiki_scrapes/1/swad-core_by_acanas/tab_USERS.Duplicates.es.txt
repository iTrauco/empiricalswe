<h1>
<a aria-hidden="true" class="anchor" href="#duplicados" id="user-content-duplicados"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Duplicados</h1>
<p>Los profesores y administradores pueden indicar que un usuario está posiblemente duplicado.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#usuarios-posiblemente-duplicados" id="user-content-usuarios-posiblemente-duplicados"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Usuarios posiblemente duplicados</h2>
<p>La opción <strong>USUARIOS &gt; Otros &gt; Duplicados</strong> lista los usuarios que han sido señalados como posiblemente duplicados. En cada fila del listado, bajo el usuario, hay dos botones:</p>
<ul>
<li>
<strong>Usuarios similares</strong>: Muestra una lista de usuarios con el mismo ID (DNI, cédula, pasaporte...) o el mismo nombre que el usuario de esta fila.</li>
<li>
<strong>No es duplicado/a</strong>: Elimina este usuario de la lista. Pulse en él cuando crea que este usuario no es un duplicado.</li>
</ul>
<h2>
<a aria-hidden="true" class="anchor" href="#usuarios-similares" id="user-content-usuarios-similares"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Usuarios similares</h2>
<p>Después de pulsar el botón <strong>Usuarios similares</strong> se muestra una lista con todos los usuarios que tienen el mismo ID o nombre que el usuario seleccionado.</p>
<p>Para cada usuario del lista se muestra la siguiente información:</p>
<ul>
<li>Datos básicos:
<ul>
<li>Foto</li>
<li>ID (DNI/cédula/pasaporte...)</li>
<li>Apellido 1</li>
<li>Apellido 2</li>
<li>Nombre</li>
<li>Institución</li>
</ul>
</li>
<li>Algunas cifras de uso de la plataforma por parte del usuario:
<ul>
<li>número de asignaturas en las que el usuario es profesor</li>
<li>número de asignaturas en las que el usuario es estudiante</li>
<li>número de archivos subidos</li>
<li>fecha y hora del último clic</li>
<li>número de clics</li>
<li>número de descargas</li>
<li>número de mensajes en foros</li>
<li>número de mensajes enviados</li>
</ul>
</li>
<li>Lista de asignaturas en las que el usuario es profesor</li>
<li>Lista de asignaturas en las que el usuario es estudiante</li>
</ul>
<p>Para cada uno de los usuarios similares mostrados en la lista se muestra un botón <strong>Eliminar cuenta de usuario</strong>. Si el usuario está marcado como posible duplicado, se muestra también un botón <strong>No es duplicado</strong>. Usando toda la información de cada usuario similar, los administradores del sistema deben decidir si alguno de los usuarios similares es realmente un duplicado y debería eliminarse.</p>
