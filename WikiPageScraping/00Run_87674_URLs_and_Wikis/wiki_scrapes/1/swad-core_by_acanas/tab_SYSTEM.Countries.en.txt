<a aria-hidden="true" class="anchor" href="#list-of-countries" id="user-content-list-of-countries"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>List of countries</h1>
<p>The <strong>SYSTEM &gt; Countries</strong> option displays a list of all countries. By default, the list is sorted by the number of users claiming to be from each country, from highest to lowest, and may also be sorted alphabetically by country name.</p>
<p>Each of the rows of the list contains the following columns:</p>
<ul>
<li>
<strong>Country</strong>: Map, name and <a href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2" rel="nofollow">alphabetic code ISO 3166-1</a>. By clicking on the header of this column ("Country") the list is sorted alphabetically by the name of the country.</li>
<li>
<strong>Users of the country</strong>: users claiming to be from the country. You can select your country in <a href="PROFILE.Record.en">PROFILE &gt; Record &gt; Edit &gt; Country</a>. By clicking on the header of this column ("Users of the country") the list is sorted by the number of users claiming to be from the country, from highest to lowest.</li>
<li>
<strong>Institutions</strong>: number of institutions in the country.</li>
<li>
<strong>Centres</strong>: number of centres in the country (centres in the institutions of the country).</li>
<li>
<strong>Degrees</strong>: number of degrees in the country (degrees in the centres of the country).</li>
<li>
<strong>Courses</strong>: number of courses in the country (courses in the degrees of the country).</li>
<li>
<strong>Teachers + students</strong>: number of users (teachers or students) in the courses of the country.</li>
</ul>
<p>At the end of the list an interactive map is displayed with the number of users in each country.</p>
<h1>
<a aria-hidden="true" class="anchor" href="#editing-countries" id="user-content-editing-countries"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Editing countries</h1>
<p>Only system administrators can edit countries, by clicking on the <em>Edit</em> icon in the upper left corner of the box.</p>
<p>First SWAD shows a form to <strong>create a new country</strong>, with the following fields:</p>
<ul>
<li>
<a href="https://en.wikipedia.org/wiki/ISO_3166-1_numeric" rel="nofollow">Numeric code ISO 3166-1</a>.</li>
<li>
<a href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2" rel="nofollow">Alphabetic code ISO 3166-1</a>.</li>
<li>Name (one entry per language): name of the country in each language.</li>
<li>WWW (one entry per language): links to the <a href="https://www.wikipedia.org/" rel="nofollow">Wikipedia</a> pages of the country in each language.</li>
</ul>
<p>Below that first box it is offered a <strong>list with all current countries</strong> where system administrators can edit the name and WWW fields and eliminate those countries that do not have any institution and any user.</p>
