<h1>
<a aria-hidden="true" class="anchor" href="#list-of-institutions" id="user-content-list-of-institutions"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>List of institutions</h1>
<p>The option <strong>COUNTRY &gt; Institutions</strong> displays a list of all institutions in the selected country. An institution is a university, college, school, academy, organization or company. By default, the list is sorted by the number of users claiming to belong to each institution, from highest to lowest, and may also be sorted by institution name.</p>
<p>Each of the rows of the list contains the following columns:</p>
<ul>
<li>
<strong>Institution</strong>: name of the institution. By clicking on the header of this column (<em>Institution</em>) the list is sorted by the name of the institution.</li>
<li>
<strong>Users of the institution</strong>: number of users claiming to belong to the institution. You can select your institution in <a href="PROFILE.Institution.en">PROFILE &gt; Record &gt; Edit my institution &gt; Institution</a>. By clicking on the header of this column (<em>Users of the institution</em>) the list is sorted by the number of users claiming to belong to the institution, from highest to lowest.</li>
<li>
<strong>Centres</strong>: number of centres in the institution.</li>
<li>
<strong>Degrees</strong>: number of degrees in the institution (degrees in the centres of the institution).</li>
<li>
<strong>Courses</strong>: number of courses in the institution (courses in the degrees of the institution).</li>
<li>
<strong>Teachers + students</strong>: number of users (teachers or students) in the courses of the institution.</li>
<li>
<strong>Status</strong>: <em>New</em>  (pending review) or <em>Active</em>.</li>
</ul>

<a aria-hidden="true" class="anchor" href="#editing-institutions" id="user-content-editing-institutions"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Editing institutions</h1>
<p>Any user can add an institution by clicking on the <em>Edit</em> icon in the upper left corner of the box or the <em>Create another institution</em> button at the end of the list.</p>
<p>First SWAD shows a form to <strong>create a new institution</strong>, with the following fields:</p>
<ul>
<li>Short name of the institution.</li>
<li>Full name of the institution.</li>
<li>WWW: full URL, including <code>http://</code> or <code>https://</code>
</li>
</ul>
<p>When a user creates a new institution, the institution goes to status <em>New</em>, which means "pending review", although it is fully usable. System administrators can visit <a href="SYSTEM.Hierarchy.en#pending">SYSTEM &gt; Hierarchy &gt; Pending</a> to check for countries with institutions that are pending review.</p>
<p>Below that box it is offered a <strong>list with all current institutions</strong> in the selected country where system administrators can edit the names and WWW fields and eliminate those institutions that do not have any centre and any user. A system administrator can also activate pending institutions after fixing errors and checking them. If the creator of the institution is not a system administrator, he/she can still continue editing the institution while it is in the <em>New</em> status.</p>
