<p>Now that your arena has been created you can place in black stained clay to have explosive blocks that launch players. This just adds a little bit of random elements to your arena.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#powerups" id="user-content-powerups"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Powerups</h2>
<p>There are currently 3 types of powerups currently. First one is Freeze. This will freeze all other players for 5 seconds. The second powerup is splash. This powerup will splash in a radius around the player. The third is swap. This will randomly change 20 blocks in the arena to your team. Powerup will spawn every 15 seconds by default. This can be set in the main config.</p>

<p>There is an update checker. If you would like to disable it. Go into the config and set update-checker to false.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#configuration" id="user-content-configuration"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Configuration</h2>
<p>The main config file has a few options. The command-whitelist option is a list. To add commands to the whitelist, add a new line after the last command whitelist and type in the command. Please note that the colorcube commands are automatically whitelisted and can not be blocked.</p>
<p>Blocks can now be configured to allow for other blocks to be paintable. To see the different types of block names check <a href="https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Material.html" rel="nofollow">here</a>.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#arena-configuration" id="user-content-arena-configuration"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Arena Configuration</h2>
<p>In version 0.7.0 and above, there is an option command to allow you to configure different options in each arena. To use the command, "/cc options &lt;id&gt; &lt;option&gt; &lt;value&gt;". Down below are the following options available:</p>
<table>
<thead>
<tr>
<th>Option</th>
<th>Type</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>perteam</td>
<td>Integer</td>
<td>Sets the number of players allowed on each team, as a result sets the total players allowed in the arena.</td>
</tr>
<tr>
<td>reward</td>
<td>Integer or Double</td>
<td>Sets the economical reward for the winning team. This value is then given to each team. This will do nothing if Vault is not installed.</td>
</tr>
<tr>
<td>pvp</td>
<td>Boolean</td>
<td>Sets whether pvp is allowed in the arena.</td>
</tr>
<tr>
<td>name</td>
<td>String</td>
<td>Sets the name of the arena which can be used to join or spectate as well as shown in the lobbysigns and game menus.</td>
</tr>
<tr>
<td>border</td>
<td>Boolean</td>
<td>Determines whether the border feature is enabled in the arena.</td>
</tr>
<tr>
<td>border-spectator-only</td>
<td>Boolean</td>
<td>Sets whether the border is only set for the spectators (true), or whether it is set for both the spectator and players (false)</td>
</tr>
<tr>
<td>border-extension</td>
<td>Integer</td>
<td>Sets the extension on the border, this will allow pushing the border farther way from the arena's edge or closer to the arena's edge.</td>
</tr>
<tr>
<td>display-scores</td>
<td>Boolean</td>
<td>Sets whether to display the scores on the side of the players screen of the team scores during the game.</td>
</tr>

</table>
