<ul>
<li>/cc join &lt;id/name&gt; - Join a game</li>
<li>/cc leave - Leave the game</li>
<li>/cc arenalist - List of all arenas</li>
<li>/cc vote - Add a vote to start the game in the users current arena.</li>
<li>/cc spectate &lt;id/name&gt; - Spectate a game.</li>
<li>/cc menuitem - Spawns a menu item into your inventory.</li>
</ul>

<ul>
<li>/cc forcestart   - Force start the specified game with the desired count down.</li>
<li>/cc disable  - Disable the specified arena.</li>
<li>/cc disable - Disables all arenas.</li>
<li>/cc enable  - Enables the specified arena.</li>
<li>/cc enable - Enables all arenas.</li>
</ul>
<h3>
<a aria-hidden="true" class="anchor" href="#admin-commands" id="user-content-admin-commands"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Admin Commands</h3>
<ul>
<li>/cc deletearena &lt;id&gt; - Deletes an arena. Can Not Be Undone.</li>
<li>/cc createarena &lt;id&gt; - Creates a new arena.</li>
<li>/cc setspawn &lt;team&gt; - Set a new or overrides a teams spawn.</li>
<li>/cc createlobby &lt;id&gt; - Creates a new lobby for a specific arena.</li>
<li>/cc reload - Reloads ColorCube and ends all current games.</li>
<li>/cc setlobbyspawn - Set the lobby spawn point inside the current lobby.</li>
<li>/cc setgloballobbyspawn - Sets the global lobby spawn for all arenas. This is the location players are returned after a game.</li>
<li>/cc removespawn &lt;team&gt; - Remove the teams spawn.</li>
<li>/cc createsign &lt;id&gt; - Creates a new arena sign for the specified arena on the sign you are looking at.</li>
<li>/cc deletesign - Deletes the arena sign that you are looking at.</li>
<li>/cc options &lt;id&gt; &lt;option&gt; &lt;value&gt; - Sets the option for the given arena. Available options: "perteam", "pvp", "reward", "name", "border", "border-extension", "border-spectator-only".</li>
<li>/cc goto &lt;id&gt; - Teleports you to the arena's lobby.</li>
</ul>
<p>Note: /cc can also be /colorcube or /colourcube</p>
