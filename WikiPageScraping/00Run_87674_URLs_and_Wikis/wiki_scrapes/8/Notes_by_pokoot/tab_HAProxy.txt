<a aria-hidden="true" class="anchor" href="#haproxycfg" id="user-content-haproxycfg"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Haproxy.cfg</h2>
<pre><code>$ sudo vim /etc/haproxy/haproxy.cfg

global
    maxconn 50000
    user haproxy
    group haproxy
    stats socket /tmp/haproxy
    node lb1
    nbproc 1
    daemon

defaults
    log global
    retries 3
    option dontlog-normal
    option splice-auto
    timeout connect 5000ms
    timeout client 5000ms
    timeout server 5000ms
    maxconn 50000

listen stats *:1936
    mode http
    stats enable
    stats uri /
    stats hide-version
    stats auth username:password

frontend frontend
    bind *:80
    bind *:443
    mode tcp
    default_backend servers

backend servers
    mode tcp
    balance roundrobin
    option httpchk HEAD / HTTP/1.1\r\nHost:\ groovey.com
    #option ssl-hello-chk
    server web01 10.0.0.2:80 check
    server web02 10.0.0.3:80 check
</code></pre>
