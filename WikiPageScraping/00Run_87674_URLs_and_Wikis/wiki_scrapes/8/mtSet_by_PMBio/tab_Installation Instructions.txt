<h2>
<a aria-hidden="true" class="anchor" href="#requirements" id="user-content-requirements"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Requirements</h2>
<p>mtSet requires Python 2.7 with scipy, h5py, numpy, pylab.</p>
<p>mtSet can use <a href="https://www.cog-genomics.org/plink2" rel="nofollow">Plink 1.9</a> to calculate the genetic relatedness matrix from bed genotype files. If Plink 1.9 is not found, a python implementation is used. We strongly recommend using Plink 1.9 for large genotype files.</p>

<p>The repository can be cloned using git</p>
<pre><code>git clone https://github.com/PMBio/mtSet.git
</code></pre>
<p>mtSet can be run from the command line using the scripts in <em>mtSet</em>/<em>bin</em> as standalone software as shown in the <a href="https://github.com/PMBio/mtSet/wiki/Tutorial">tutorial</a>.</p>
