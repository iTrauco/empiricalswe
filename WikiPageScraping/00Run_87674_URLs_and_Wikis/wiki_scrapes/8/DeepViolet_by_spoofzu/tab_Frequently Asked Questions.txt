          <p>How do I report bugs?
Run the command line version of DeepViolet with the following command line options.
Step 1) java -jar dvCMD.jar -d -d2 -serverurl <a href="https://hostname.com/" rel="nofollow">https://hostname.com/</a> &gt; hostname.txt
Step 2) Open a New Issue GitHub issue in the DeepViolet project.
Step 3) Include your explanation of the problem. Attach the output from Step 1 to your issue.
*If DeepViolet will not run at all then include: description of the problem, version of Java, and anything else you think could be helpful.</p>
<p>How do I suggest features?
Open a New Issue and include your suggestion.</p>
