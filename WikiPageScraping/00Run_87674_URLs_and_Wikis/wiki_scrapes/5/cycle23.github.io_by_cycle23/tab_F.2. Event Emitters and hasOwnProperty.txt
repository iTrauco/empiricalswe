        <div class="markdown-body">
          <p>The <a href="https://github.com/Reactive-Extensions/RxJS/blob/master/doc/howdoi/eventemitter.md">Event Emitters</a> example makes use of <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/hasOwnProperty" rel="nofollow">hasOwnProperty</a> and does so using an anonymous function definition to be as general as possible (see later in that definition).</p>
<p>A deeper explanation for this shows up in this <a href="http://stackoverflow.com/questions/12017693/why-use-object-prototype-hasownproperty-callmyobj-prop-instead-of-myobj-hasow" rel="nofollow">StackOverflow hasOwnProperty article</a>.</p>
<p>The idea in the event emitters example appears to be to make sure dispose is not called for all properties down an inheritance chain, just the immediate items.</p>
