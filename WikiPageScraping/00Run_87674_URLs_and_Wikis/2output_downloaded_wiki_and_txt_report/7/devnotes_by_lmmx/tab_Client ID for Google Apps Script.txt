          <p>This won't exist automatically for scripts, and scripts don't exist in the Developer console projects, though if you go to <code>Tools</code> &gt; <code>Script Editor</code> &gt; <code>Resources</code> &gt; <code>Advanced Google Services</code> the link there will lead to a project page for the script, with a link to <code>Credentials</code> where you can create a client ID.</p>
<p>The client ID is a web address string, the app ID is the first portion of the client ID (separated by a hyphen).</p>
<p>The example given on the Drive REST API <a href="https://developers.google.com/drive/web/integrate-open" rel="nofollow">docs for the <code>Open</code> command</a> are</p>
<pre>clientId: 1234567890-abcdefghijklmnopqrstuvwxyz.apps.googleusercontent.com
appId:    1234567890</pre>
