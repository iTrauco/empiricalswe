          <p>Like Biostars is the StackExchange/StackOverflow for bioinformatics, there needs to be a (more rigorous) community-maintained site for</p>
<ul>
<li>listing bioinformatics tools</li>
<li>listing bioinformatics databases
<ul>
<li>NAR's would be a starting point, but also linking them</li>
</ul>
</li>
<li>relating them as "alternatives to" one another - i.e. assigning specific tasks to each</li>
<li>recording the publications associated with each</li>
</ul>
<p><a href="http://alternativeto.net/" rel="nofollow">alternativeto.net</a> is the only thing I can think of doing something similar</p>
<p>See also: <a href="http://www.biocurator.org/" rel="nofollow">Society for Biocuration</a>, <a href="https://github.com/ialbert/biostar-central">biostar-central</a> repository on GitHub (for a forum about disputes?)</p>
<ul>
<li>Not really a Wiki... that's what Wikipedia's for. Talk pages on discrete 'tasks'</li>
</ul>
<p>Should probably just link to Wikipedia, improve Wikipedia entries, then parse those through Wikipedia's API for things like publication records.</p>
