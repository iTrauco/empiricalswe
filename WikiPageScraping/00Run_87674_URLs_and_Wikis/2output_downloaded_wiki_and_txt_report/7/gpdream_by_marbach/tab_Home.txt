          <p><strong>GP-DREAM is a growing collection of gene network inference tools that can be transparently launched through a web interface</strong>. The tools include best-performing methods from the <a href="http://dreamchallenges.org/" rel="nofollow"><strong>DREAM Network Inference Challenge</strong></a> as well as other methods contributed by the community. The GP-DREAM server is powered by <a href="http://www.broadinstitute.org/cancer/software/genepattern/" rel="nofollow"><strong>GenePattern</strong></a>. Tools can also be used as standalone command-line applications.</p>

<li>
<strong>Give it a go</strong>: <a href="http://dream.broadinstitute.org" rel="nofollow"><strong>dream.broadinstitute.org</strong></a>
</li>
