          <p>The following is a brief introduction to the transforms provided by FetLife Maltego.</p>
<ul>
<li>AliasToFetLifeProfile
<ul>
<li>Tests for the existence of a FetLife user with the given alias (nickname).</li>
</ul>
</li>
<li>To FetLife Friends
<ul>
<li>Extracts the user's Friends list.</li>
</ul>
</li>
<li>To URL [Links on this FetLife page]
<ul>
<li>Extracts URLs pointing to external websites present on the associated FetLife page.</li>
</ul>
</li>
<li>To Upcoming FetLife Events
<ul>
<li>Extracts the upcoming FetLife Events the user has RSVP'ed to.</li>
</ul>
</li>
<li>To Entities By Crawl
<ul>
<li>Extracts various recognized entities (FetLife users, events, email addresses, etc.) present on the associated FetLife page.</li>
</ul>
</li>
<li>convertToLocation
<ul>
<li>This transform returns the reported location information for the event or profile.</li>
</ul>
</li>
<li>To Profile Details
<ul>
<li>This transform populates an <code>Affiliation - FetLife</code> entity with information from the user's FetLife profile.</li>
</ul>
</li>
</ul>
