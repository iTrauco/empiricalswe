          <p>Connecting to router's ftp server via finder is very easy:</p>
<p>Open a new finder window. Then, select the Go menu in the menu toolbar.</p>
<p>Select connect to server, note that you can directly access this option with cmd+k.</p>
<p>A window like this one will display:</p>
<img data-canonical-src="http://s22.postimg.org/b8xto6m0h/third.png" src="https://camo.githubusercontent.com/4e52bb87f0a2c0269ccc54d85296890c7b88a18d/687474703a2f2f7332322e706f7374696d672e6f72672f623878746f366d30682f74686972642e706e67"/>
<p>In "server direction" type ftp://10.0.0.1, 10.0.0.1 is the Netbeast router gateway IP direction in the netbeast-xy network.</p>
<p>The only thing left is to hit "Connect". You are ready to access all your router's files!</p>
<p>Any doubt reach us at @netbeast_co (twitter) or staff [at] netbeast.co.</p>
