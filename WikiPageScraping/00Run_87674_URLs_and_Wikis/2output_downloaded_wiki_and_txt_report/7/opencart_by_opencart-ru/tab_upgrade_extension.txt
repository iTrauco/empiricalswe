          <h1>
<a aria-hidden="true" class="anchor" href="#%D0%90%D0%B4%D0%B0%D0%BF%D1%82%D0%B0%D1%86%D0%B8%D1%8F-%D0%BC%D0%BE%D0%B4%D1%83%D0%BB%D0%B5%D0%B9-opencart-15x-%D0%BA-opencart-20x" id="user-content-Адаптация-модулей-opencart-15x-к-opencart-20x"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Адаптация модулей OpenCart 1.5.x к OpenCart 2.0.x</h1>
<p>В OpenCart, как и в любом другом программном продукте, смена мажорной версии означает потерю совместимости. Модули под версии 1.5.x больше не подходят для OpenCart 2, их придется адаптировать. Итак, что нужно менять в старых дополнениях.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#%D0%A0%D0%B5%D0%BD%D0%B4%D0%B5%D1%80%D0%B8%D0%BD%D0%B3-%D1%88%D0%B0%D0%B1%D0%BB%D0%BE%D0%BD%D0%B0" id="user-content-Рендеринг-шаблона"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Рендеринг шаблона</h2>
<p>Раньше для передачи данных в шаблон надо было менять поле <strong>$data</strong> контроллера, сам шаблон брался из поля <strong>$title</strong>, а вывод осуществлял метод <strong>render</strong>:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">data</span>[<span class="pl-s"><span class="pl-pds">'</span>title<span class="pl-pds">'</span></span>] <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">language</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>title<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"> </span>
<span class="pl-s1"><span class="pl-k">if</span> (<span class="pl-c1">file_exists</span>(<span class="pl-c1">DIR_TEMPLATE</span> <span class="pl-k">.</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">config</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>config_template<span class="pl-pds">'</span></span>) <span class="pl-k">.</span> <span class="pl-s"><span class="pl-pds">'</span>/template/module/example.tpl<span class="pl-pds">'</span></span>)) {</span>
<span class="pl-s1">	<span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">template</span> <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">config</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>config_template<span class="pl-pds">'</span></span>) <span class="pl-k">.</span> <span class="pl-s"><span class="pl-pds">'</span>/template/module/example.tpl<span class="pl-pds">'</span></span>;</span>
<span class="pl-s1">} <span class="pl-k">else</span> {</span>
<span class="pl-s1">	<span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">template</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">'</span>default/template/module/example.tpl<span class="pl-pds">'</span></span>;</span>
<span class="pl-s1">}</span>
<span class="pl-s1"> </span>
<span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span>render();</span></pre></div>
<p>, то теперь используются локальные переменные метода контроллера:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$data</span> <span class="pl-k">=</span> <span class="pl-c1">array</span>();</span>
<span class="pl-s1"><span class="pl-smi">$data</span>[<span class="pl-s"><span class="pl-pds">'</span>title<span class="pl-pds">'</span></span>] <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">language</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>title<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"> </span>
<span class="pl-s1"><span class="pl-k">if</span> (<span class="pl-c1">file_exists</span>(<span class="pl-c1">DIR_TEMPLATE</span> <span class="pl-k">.</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">config</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>config_template<span class="pl-pds">'</span></span>) <span class="pl-k">.</span> <span class="pl-s"><span class="pl-pds">'</span>/template/module/example.tpl<span class="pl-pds">'</span></span>)) {</span>
<span class="pl-s1">	<span class="pl-smi">$template</span> <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">config</span><span class="pl-k">-&gt;</span>get(<span class="pl-s"><span class="pl-pds">'</span>config_template<span class="pl-pds">'</span></span>) <span class="pl-k">.</span> <span class="pl-s"><span class="pl-pds">'</span>/template/module/example.tpl<span class="pl-pds">'</span></span>;</span>
<span class="pl-s1">} <span class="pl-k">else</span> {</span>
<span class="pl-s1">	<span class="pl-smi">$template</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">'</span>default/template/module/example.tpl<span class="pl-pds">'</span></span>;</span>
<span class="pl-s1">}</span>
<span class="pl-s1"> </span>
<span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">response</span><span class="pl-k">-&gt;</span>setOutput(<span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">load</span><span class="pl-k">-&gt;</span>view(<span class="pl-smi">$template</span>, <span class="pl-smi">$data</span>));</span></pre></div>
<p>Иначе теперь стали выводиться хедеры, футеры и другие подчиненные (children) блоки. Например, для страниц в админке было так:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">children</span> <span class="pl-k">=</span> <span class="pl-c1">array</span>(</span>
<span class="pl-s1">	<span class="pl-s"><span class="pl-pds">'</span>common/header<span class="pl-pds">'</span></span>,</span>
<span class="pl-s1">	<span class="pl-s"><span class="pl-pds">'</span>common/footer<span class="pl-pds">'</span></span></span>
<span class="pl-s1">);</span></pre></div>
<p>, а стало так:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$data</span>[<span class="pl-s"><span class="pl-pds">'</span>header<span class="pl-pds">'</span></span>] <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">load</span><span class="pl-k">-&gt;</span>controller(<span class="pl-s"><span class="pl-pds">'</span>common/header<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-smi">$data</span>[<span class="pl-s"><span class="pl-pds">'</span>column_left<span class="pl-pds">'</span></span>] <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">load</span><span class="pl-k">-&gt;</span>controller(<span class="pl-s"><span class="pl-pds">'</span>common/column_left<span class="pl-pds">'</span></span>);</span>
<span class="pl-s1"><span class="pl-smi">$data</span>[<span class="pl-s"><span class="pl-pds">'</span>footer<span class="pl-pds">'</span></span>] <span class="pl-k">=</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">load</span><span class="pl-k">-&gt;</span>controller(<span class="pl-s"><span class="pl-pds">'</span>common/footer<span class="pl-pds">'</span></span>);</span></pre></div>
<p>Обратите внимание, что в админке OpenCart 2 левая колонка выделена в отдельный шаблон.</p>
<p>По-другому теперь осуществляется переадрессация (redirect). Было:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span>redirect(<span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">url</span><span class="pl-k">-&gt;</span>link(<span class="pl-s"><span class="pl-pds">'</span>extension/feed<span class="pl-pds">'</span></span>, <span class="pl-s"><span class="pl-pds">'</span>token=<span class="pl-pds">'</span></span> <span class="pl-k">.</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">session</span><span class="pl-k">-&gt;</span><span class="pl-smi">data</span>[<span class="pl-s"><span class="pl-pds">'</span>token<span class="pl-pds">'</span></span>], <span class="pl-s"><span class="pl-pds">'</span>SSL<span class="pl-pds">'</span></span>));</span></pre></div>
<p>, а стало:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">response</span><span class="pl-k">-&gt;</span>redirect(<span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">url</span><span class="pl-k">-&gt;</span>link(<span class="pl-s"><span class="pl-pds">'</span>extension/feed<span class="pl-pds">'</span></span>, <span class="pl-s"><span class="pl-pds">'</span>token=<span class="pl-pds">'</span></span> <span class="pl-k">.</span> <span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">session</span><span class="pl-k">-&gt;</span><span class="pl-smi">data</span>[<span class="pl-s"><span class="pl-pds">'</span>token<span class="pl-pds">'</span></span>], <span class="pl-s"><span class="pl-pds">'</span>SSL<span class="pl-pds">'</span></span>));</span></pre></div>

<p>В контроллере модуля оплаты метод <strong>index</strong> был protected</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-k">class</span> <span class="pl-en">ControllerPaymentPmethod</span> <span class="pl-k">extends</span> <span class="pl-e">Controller</span> {</span>
<span class="pl-s1">	<span class="pl-k">protected</span> <span class="pl-k">function</span> <span class="pl-en">index</span>() {</span></pre></div>
<p>, стал public:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-k">class</span> <span class="pl-en">ControllerPaymentPmethod</span> <span class="pl-k">extends</span> <span class="pl-e">Controller</span> {</span>
<span class="pl-s1">	<span class="pl-k">public</span> <span class="pl-k">function</span> <span class="pl-en">index</span>() {</span></pre></div>
<p>Платежные модули при успешной оплате обычно меняют статус заказа. Раньше статус менялся так:</p>
<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">model_checkout_order</span><span class="pl-k">-&gt;</span>confirm(<span class="pl-smi">$order_id</span>, <span class="pl-smi">$order_status_id</span>, <span class="pl-s"><span class="pl-pds">'</span>Pmethod<span class="pl-pds">'</span></span>);</span></pre></div>

<div class="highlight highlight-text-html-php"><pre><span class="pl-s1"><span class="pl-smi">$this</span><span class="pl-k">-&gt;</span><span class="pl-smi">model_checkout_order</span><span class="pl-k">-&gt;</span>addOrderHistory(<span class="pl-smi">$order_id</span>, <span class="pl-smi">$order_status_id</span>);</span></pre></div>
