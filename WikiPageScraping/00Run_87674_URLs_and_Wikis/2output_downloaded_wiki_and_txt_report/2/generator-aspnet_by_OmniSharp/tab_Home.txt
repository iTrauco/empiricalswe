          <p>Welcome to the project wiki for <code>generator-aspnet</code>! The <code>generator-aspnet</code> is a generator for <a href="http://yeoman.io/" rel="nofollow">yo</a>, a command-line scaffolding system built on <a href="http://nodejs.org/" rel="nofollow">node.js</a>.</p>
<p>With this generator, it's possible to scaffold projects and files for ASP.Net vNext on Windows, Mac OS X, and Linux -- streamlining development of .net projects with your favorite text editors.</p>

</h3>
<p><a href="https://github.com/xtianus79/generator-aspnet"><code>generator-aspnet-xtianus</code></a> is an extension of OmniSharp/generator-aspnet that comes with a special Foundation 5 SASS/SCSS framework ready out of the box with wiredep &amp; other grunt tasks for advanced front-end development. Look for =&gt; <a href="https://github.com/xtianus79/generator-aspnet/blob/master/templates/projects/foundation5/README.md"><code>Starter Web Application - Foundation 5</code></a>. The other goal of this generator is to provide alternative templates to the traditional ASP.NET Visual Studio templates. More templates will become housed under this fork in the near future.  Feel free to participate and learn more about <a href="https://github.com/xtianus79/generator-aspnet">`generator-aspnet-xtianus</a>.</p>
<p>If you are working on a related generator please <a href="https://github.com/OmniSharp/generator-aspnet/issues/new">open an issue</a> to let us know about it so that we can add it to the list.</p>
