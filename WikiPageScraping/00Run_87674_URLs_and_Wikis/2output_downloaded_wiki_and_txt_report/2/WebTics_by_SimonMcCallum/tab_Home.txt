          <p>Webtics is a game metrics and analysis system designed for SME (small to medium enterprises) game developers who want to have a light weight in-house opensource game metrics recording system.</p>
<p>This system provides the database back-end and API front-end for a logging system.  This system is focused on logging events generated by the game when the developer chooses to log an event.</p>
<p>This wiki is contains the API description, and a discussion of the purpose of Webtics.</p>
