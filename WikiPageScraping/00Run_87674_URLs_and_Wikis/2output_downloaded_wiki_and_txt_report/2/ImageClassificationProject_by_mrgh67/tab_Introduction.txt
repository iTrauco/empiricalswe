<a aria-hidden="true" class="anchor" href="#about-the-dataset" id="user-content-about-the-dataset"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>About the dataset:</h1>
<p>The CIFAR-10 dataset consists of 60000 color images.</p>
<p>Each color image is tiny, being only 32 pixels by 32 pixels in dimension.</p>
<p>The database of images is divided up into a training set of 50000 images and a test set of 10000 images.</p>
<p>All the images have been hand scored to belong to 1 of 10 classes:</p>
<ol>
<li>airplane</li>
<li>automobile</li>
<li>bird</li>
<li>cat</li>
<li>deer</li>
<li>dog</li>
<li>frog</li>
<li>horse</li>
<li>ship</li>
<li>truck</li>
</ol>
<p>There are 5000 training images for each class, and 1000 test images for each class.</p>

<a aria-hidden="true" class="anchor" href="#wider-learning-objectives" id="user-content-wider-learning-objectives"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Wider Learning Objectives</h3>
<p>Whilst it would have been extremely rewarding to have achieved a high success rate in classifying the images the broader aim of the project included the following learning objectives:</p>
<ul>
<li>Become familiar with Python image processing tools</li>
<li>Investigate pre-processing of the images looking for distinctive features</li>
<li>Investigate 2 dimensional Fourier transforms</li>
<li>Experience first hand what it is like to handle image data and the challenges required to manipulate images into data features that can be used within a classification paradigm</li>
<li>Acquire some experience using decision trees and related classifiers</li>
</ul>
