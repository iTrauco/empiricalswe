<div>
<div>
<p>We distinguish two client-side integration levels to test on:
* Client-side module test: Tests against the controller.
* Client-side integration test: Tests against a browser.</p>
</div>
<div>
<p>Both integration levels do not include the real server. Instead of that, the server is replaced by a mock.
For more information about OASP integration levels read the <a href="https://github.com/oasp/oasp4j/wiki/guide-testing">testing page in the OASP4J wiki</a>.</p>
