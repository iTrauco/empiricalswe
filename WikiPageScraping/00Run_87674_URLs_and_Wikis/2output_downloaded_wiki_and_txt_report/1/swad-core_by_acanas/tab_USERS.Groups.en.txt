<h2>
<a aria-hidden="true" class="anchor" href="#select-groups" id="user-content-select-groups"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Select groups</h2>
<p>If the current course has groups, in all the options where students are listed you can change the group selection and then click on <strong>Update students</strong> to update the list of students. By default the groups to which you belong are selected. You can join other groups in <strong>USERS &gt; Groups</strong>. When selecting groups, you can click on <strong>All groups</strong> to activate all or deactivate all (if enabled).</p>
<p>The options where you can change the group selection are:</p>
<ul>
<li><a href="ASSESSMENT.Tests.en#test-results">ASSESSMENT &gt; Tests &gt; Test results</a></li>
<li><a href="FILES.Homework.en#for-teachers">FILES &gt; Homework</a></li>
<li><a href="USERS.Students.en">USERS &gt; Students</a></li>
<li>
<a href="USERS.Attendance.en">USERS &gt; Attendance</a> &gt; Show one event</li>
<li><a href="USERS.Attendance.en#attendance-list">USERS &gt; Attendance &gt; List attendance</a></li>
<li><a href="MESSAGES.Write.en">MESSAGES &gt; Write</a></li>
<li><a href="MESSAGES.Email.en">MESSAGES &gt; Email</a></li>
<li><a href="STATS.Visits.en#visits-to-course">STATS &gt; Logins &gt; Visits to course</a></li>
</ul>
<h2>
<a aria-hidden="true" class="anchor" href="#join-groups" id="user-content-join-groups"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Join groups</h2>
<p>You can join (or delete from) groups by selecting them in <strong>USERS &gt; Groups</strong> and then clicking on the button under the group list labelled as:</p>
<ul>
<li>
<em>Enrol in groups</em>, if you are not already enrolled in any group</li>
<li>
<em>Change my groups</em>, if you are already enrolled in a group</li>
</ul>
<p>If you are a student in the course, you can only register in (or delete from) open groups, marked with an open green padlock.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#create-groups" id="user-content-create-groups"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Create groups</h2>
<p>To create or edit groups in a course you must be logged as teacher in that course. Go to <strong>USERS &gt; Groups</strong> and click on the <em>Edit</em> icon (pencil) in the upper left corner of the <em>Groups</em> box.</p>
<ul>
<li>First <strong>create group types</strong> (USERS &gt; Groups &gt; Edit). Examples: <code>Lectures</code>, <code>Practicals</code>, <code>Lab</code>, <code>Tutorials</code>.
<ul>
<li>For each type of group, you must indicate the type of enrolment:
<ul>
<li>Mandatory or voluntary</li>
<li>Single or multiple</li>
</ul>
</li>
<li>You can also set a time to automatically open the groups of a type.</li>
</ul>
</li>
<li>Once types of group are created, <strong>create groups</strong> (USERS &gt; Groups &gt; Edit). Examples: <code>A</code>, <code>B</code>, <code>1</code>, <code>2</code>, <code>Mondays 13-15h</code>.
<ul>
<li>You can limit the number of students.</li>
<li>You can open groups for the students to sign up.</li>
<li>You can enable file areas for groups (for <a href="FILES.Documents.en">documents</a>, <a href="FILES.Private.en">private files</a> or <a href="FILES.Shared.en">shared files</a>).</li>
</ul>
</li>
<li>Once the groups are created, you should <strong>join</strong> some of them:
<ul>
<li>To set your favourite list of default groups to list students.</li>
<li>To be able to enrol students into groups.</li>
</ul>
</li>
</ul>

<a aria-hidden="true" class="anchor" href="#edit-groups" id="user-content-edit-groups"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Edit groups</h2>
<p>Group types and groups already created can be edited by teachers at any time by clicking the <em>Edit</em> icon (pencil) in the upper left corner of <strong>USERS &gt; Groups</strong>.</p>
