<p>The <strong>DEGREE &gt; Information</strong> option displays a table with basic information about the selected degree, automatically generated, with the following items:</p>
<ul>
<li>
<strong>Logo</strong>. Logo of the degree. Editable by degree administrators, centre administrators, institution administrators and system administrators:
<ul>
<li>Mandatory file format: <em>png</em>.</li>
<li>Recommended background: transparent.</li>
<li>Mandatory image resolution: 64x64 pixels.</li>
</ul>
</li>
<li>
<strong>Centre</strong>. Centre to which the degree belongs. Editable by institution administrators and system administrators.</li>
<li>
<strong>Degree</strong>. Full name of the degree. Editable by centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Short name</strong>. Short name of the degree. Editable by centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Web</strong>. Link to the external website of the degree. Editable by degree administrators, centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Shortcut</strong>. Direct link to the degree inside SWAD. You can copy this link and put it in an external web page to enter directly to the degree.</li>
<li>
<strong>Courses</strong>. Current number of courses in the degree.</li>
<li>
<strong>Teachers</strong>. Current number of teachers registered in the courses of the degree.</li>
<li>
<strong>Students</strong>. Current number of students registered in the courses of the degree.</li>
</ul>
<p>The <em>Print</em> icon at the top left of the box shows a printable version of this table, including a <a href="https://en.wikipedia.org/wiki/QR_code" rel="nofollow">QR code</a> associated to the degree. The <em>Logo</em> icon at the top left of the box allows uploading a new logo for the degree.</p>
