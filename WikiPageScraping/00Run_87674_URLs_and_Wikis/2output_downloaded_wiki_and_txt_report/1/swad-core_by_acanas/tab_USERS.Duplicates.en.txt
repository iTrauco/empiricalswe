<p>Teachers and administrators can report that a user is possibly duplicated.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#possibly-duplicate-users" id="user-content-possibly-duplicate-users"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Possibly duplicate users</h2>
<p>The option <strong>USERS &gt; Others &gt; Duplicates</strong> lists the users who have been reported as possibly duplicates. In each row in the list, under the user, there are two buttons:</p>
<ul>
<li>
<strong>Similar users</strong>: Displays a list of users with the same ID (identity number/passport) or the same name as the user in this row.</li>
<li>
<strong>Not duplicated</strong>: Removes this user from the list. Click on it when you think that this user is not duplicated.</li>
</ul>
<h2>
<a aria-hidden="true" class="anchor" href="#similar-users" id="user-content-similar-users"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Similar users</h2>
<p>After clicking on the <strong>Similar users</strong> button a list is displayed with all the users that share the ID or the name with the selected user.</p>
<p>For each user in the list the following user information is displayed:</p>
<ul>
<li>Basic data:
<ul>
<li>Photo</li>
<li>ID (identity number/passport)</li>
<li>Surname 1</li>
<li>Surname 2</li>
<li>First name</li>
<li>Institution</li>
</ul>
</li>
<li>Some figures of use of the platform by the user:
<ul>
<li>number of courses in which the user is a teacher</li>
<li>number of courses in which the user is a student</li>
<li>number of files uploaded</li>
<li>date and time of his/her first click</li>
<li>number of clicks</li>
<li>number of downloads</li>
<li>number of posts in forums</li>
<li>number of messages sent</li>
</ul>
</li>
<li>List of courses in which the user is a teacher</li>
<li>List of courses in which the user is a student</li>
</ul>
<p>For each of the similar users shown in the list an <strong>Eliminate user account</strong> button is displayed. If the user is marked as a possible duplicate, a <strong>Not duplicated</strong> button is displayed. Using all the information from each similar user, system administrators must decide if any of the similar users are actually duplicates and should be removed.</p>
