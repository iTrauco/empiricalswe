<h2>
<a aria-hidden="true" class="anchor" href="#servidor" id="user-content-servidor"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Servidor</h2>
<p>El núcleo de SWAD es un <a href="https://es.wikipedia.org/wiki/Interfaz_de_entrada_com%C3%BAn" rel="nofollow">CGI</a> programado en <a href="https://es.wikipedia.org/wiki/C_(lenguaje_de_programaci%C3%B3n)" rel="nofollow">C</a> que incluye casi toda la funcionalidad de la plataforma. El núcleo se complementa con algunos programas externos como el módulo de procesamiento de fotografías y el módulo de chat. El servidor se ejecuta en un sistema <a href="https://es.wikipedia.org/wiki/GNU/Linux" rel="nofollow">GNU/Linux</a> con <a href="https://es.wikipedia.org/wiki/Servidor_HTTP_Apache" rel="nofollow">Apache</a> y una base de datos <a href="https://es.wikipedia.org/wiki/MySQL" rel="nofollow">MySQL</a> o <a href="https://es.wikipedia.org/wiki/MariaDB" rel="nofollow">MariaDB</a>.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#clientes" id="user-content-clientes"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Clientes</h2>
<p>Al tratarse de una aplicación web, el cliente puede ser cualquier navegador web moderno. Para usar el chat es necesario haber instalado <a href="https://www.java.com/es/download/" rel="nofollow">Java Runtime Environment</a>. Además del cliente web, hay una aplicación de M-learning para dispositivos Android llamada <a href="https://openswad.org/swadroid/" rel="nofollow">SWADroid</a>, que implementa algunas de las características más usadas de la versión web.</p>
