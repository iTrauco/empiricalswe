<h1>
<a aria-hidden="true" class="anchor" href="#information-about-course" id="user-content-information-about-course"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Information about course</h1>
<p>The <strong>COURSE &gt; Information</strong> option displays information about the selected course distributed in two boxes:</p>
<ol>
<li>
<strong><a href="#basic-information">Basic information</a></strong> about the course, automatically generated.</li>
<li>
<strong><a href="#textual-information">Textual information</a></strong>, editable by teachers.</li>
</ol>
<h2>
<a aria-hidden="true" class="anchor" href="#basic-information" id="user-content-basic-information"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Basic information</h2>
<p>The <strong>first box shows a table with basic information</strong> about the course, automatically generated, with the following items:</p>
<ul>
<li>
<strong>Degree</strong>.  Editable by centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Course</strong>. Full name of the course.</li>
<li>
<strong>Short name</strong>. Short name of the course.</li>
<li>
<strong>Year or semester</strong>. Editable by teachers, degree administrators, centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Institutional code</strong>. Optional. Editable by teachers, degree administrators, centre administrators, institution administrators and system administrators.</li>
<li>
<strong>Internal code</strong>. Numerical code automatically generated when the course was created.</li>
<li>
<strong>Shortcut</strong>. Direct link to the course inside SWAD. You can copy this link and put it in an external web page to enter directly to the course.</li>
<li>
<strong>Teachers</strong>. Current number of teachers registered in the course.</li>
<li>
<strong>Students</strong>. Current number of students registered in the course.</li>
<li>
<strong>Indicators</strong>. Number of quality indicators (0 to 5) fulfilled by the course, linking to <a href="STATS.Indicators.en">STATS &gt; Indicators</a>.</li>
</ul>
<p>The <em>Print</em> icon at the top left of the box shows a printable version of this table, including a <a href="https://en.wikipedia.org/wiki/QR_code" rel="nofollow">QR code</a> associated to the course.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#textual-information" id="user-content-textual-information"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Textual information</h2>
<p>The <strong>second box shows a wider textual information</strong> about the course, editable by teachers. Teachers can force students to read this information by checking the <em>Force students to read this information</em> checkbox.</p>
<h3>
<a aria-hidden="true" class="anchor" href="#edit" id="user-content-edit"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Edit</h3>
<p>To edit the information about the course, you must click on the <em>Edit</em> icon (pencil) at the top left of the box. A new box titled <strong>Source of information</strong> is displayed in which you can choose one of the following options:</p>
<ul>
<li>
<strong>Do not give any information</strong>.</li>
<li>
<strong>Plain text</strong>. A <a href="https://en.wikipedia.org/wiki/Plain_text" rel="nofollow">plain text</a> editor. You can write inline math formulas with <code>\(LaTeX\)</code> or centred equations with <code>$$LaTeX$$</code> or <code>\[LaTeX\]</code>.</li>
<li>
<strong>Rich text</strong>. A rich text editor. In this case you can use <a href="http://pandoc.org/MANUAL.html#pandocs-markdown" rel="nofollow">Pandoc's Markdown</a> to format the text. You can also write inline math formulas with <code>$LaTeX$</code> or centred equations with <code>$$LaTeX$$</code>.</li>
<li>
<strong>Web page hosted in SWAD</strong>. A page uploaded to SWAD server. A form is shown to upload either a single HTML file if your page contains no images, or a ZIP file containing an <code>index.html</code> or <code>index.htm</code> HTML file and a folder with image files.</li>
<li>
<strong>Link to web page hosted on another server</strong>. You have to write a URL, that is, a link to a page or external document hosted on another server.</li>
</ul>
