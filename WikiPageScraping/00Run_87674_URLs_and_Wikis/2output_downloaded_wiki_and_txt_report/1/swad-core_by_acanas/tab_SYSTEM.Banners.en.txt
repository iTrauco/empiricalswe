<p>The <strong>SYSTEM &gt; Links &gt; Banners</strong> option lists the advertising banners that are displayed in the right column of the page. These banners change randomly every few minutes.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#edit" id="user-content-edit"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Edit</h2>
<p>System administrators can edit banners by clicking on the <em>Edit</em> icon (pencil) in the upper left corner.</p>
<p>When you click on <em>Edit</em> two boxes appear:</p>
<ul>
<li>
<strong>New banner</strong>. The first box allows you to <strong>add a new banner</strong> by typing the following columns and then clicking on the <em>Create banner</em> button:
<ul>
<li>
<strong>Short name</strong>. This name will be displayed if the image can not be loaded.</li>
<li>
<strong>Full name</strong>. This name will be displayed when the user places the mouse pointer over the banner image.</li>
<li>
<strong>Image</strong>. The image will be displayed in a resolution of 120x40 pixels and should have an aspect ratio of 3:1 and a minimum resolution of 120x40 pixels (240x80 pixels is recommended). The system administrator must upload the image manually to the public banners directory, indicated in the configuration file <code>swad_config.h</code>.</li>
<li>
<strong>WWW</strong>. URL to which the link will point.</li>
</ul>
</li>
<li>
<strong>Banners</strong>. The second box allows you to <strong>edit the existing banners</strong>. You can change any of the four previous columns.</li>
</ul>
<p>To <strong>remove</strong> a banner, click on the <em>Remove</em> icon (red cross). No confirmation is required.</p>
<p>You can also <strong>hide</strong> visible banners by clicking on the <em>Hide</em> icon (eye), and <strong>unhide</strong> hidden banners by clicking on the <em>Unhide</em> icon (strikethrough eye).</p>
