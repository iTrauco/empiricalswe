<ul>
<li>cc.arena.join.&lt;arena id&gt; - Gives the player the permission to join that arena</li>
<li>cc.arena.join.* - Gives the player the permission to join all arenas</li>
<li>cc.arena.vote - Gives the player permission to vote in an arena.</li>
<li>cc.arena.spectate. - Gives player the permission to spectate arena.</li>
<li>cc.arena.spectate.* - Gives player the permission to spectate all arenas</li>
<li>cc.lobby.menu - Perms to open menu and give menu item via command.</li>
</ul>
<h3>
<a aria-hidden="true" class="anchor" href="#admin-permissions" id="user-content-admin-permissions"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Admin Permissions:</h3>
<ul>
<li>cc.admin.* - Grants all cc.admin. permissions</li>
<li>cc.admin.reload - Grants /cc reload &lt;all | Settings| Games&gt;</li>
<li>cc.admin.setgloballobbyspawn - Grants the ability to set the global lobby spawn</li>
<li>cc.admin.forcestart - Grants forcestart command</li>
<li>cc.arena.* - Grants all cc.arena permissions</li>
<li>cc.arena.create - Create arenas</li>
<li>cc.arena.delete - Delete arenas</li>
<li>cc.arena.disable - Disable arenas</li>
<li>cc.arena.enable - Enable arenas</li>
<li>cc.arena.setspawn - Set spawn points inside the arena</li>
<li>cc.arena.removespawn - Reset all spawn points inside arena</li>
<li>cc.arena.goto - Goto Command permission</li>
<li>cc.arena.options - Set options for the arena.</li>
<li>cc.lobby.* - All lobby permissions</li>
<li>cc.lobby.create - Create lobbys for arena</li>
<li>cc.lobby.setlobbyspawn - Set lobby spawns</li>
<li>cc.lobby.sign.create - Perms for creating lobby sign</li>
<li>cc.lobby.sign.delete - Perms to delete lobby sign</li>
<li>cc.* - All ColorCube permissions</li>
</ul>
