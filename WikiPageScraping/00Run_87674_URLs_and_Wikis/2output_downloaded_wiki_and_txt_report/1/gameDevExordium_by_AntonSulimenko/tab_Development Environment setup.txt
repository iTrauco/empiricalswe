          <p>Project is created with cmake. You can get it from <a href="http://www.cmake.org/download/" rel="nofollow">here</a></p>
<p>CMake uses OGRE_HOME System Environment variable to find Ogre SDK directory and generates project's dependencies paths.</p>
<p>If it is not specified or you want to change path to Ogre you can change it in CMake via change the OGRE_PATH option.</p>
<hr/>
<p>OGRE SDK is required to build Exordium.</p>
<p>Sdk can be downloaded from <a href="http://www.ogre3d.org/download/sdk" rel="nofollow">here</a>.</p>
<p>If any of prebuild version is not suitable for you (platform etc) you can download source and build on your own.</p>
