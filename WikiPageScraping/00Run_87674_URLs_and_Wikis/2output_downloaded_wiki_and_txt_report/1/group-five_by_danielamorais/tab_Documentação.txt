          <p><strong>Requisitos Funcionais</strong></p>
<ol>
<li>Comunicação com sensores
O software deve manter relação com os sensores para detectar presença e/ou alteração no posicionamento do sensor do terreno.</li>
<li>Acesso remoto
Proporcionar para o usuário acesso ao sistema via WEB.</li>
<li>Gerar relatórios
Permitir a geração de relatórios e gráficos sobre o desempenho dos sensores.</li>
<li>Backup
Realizar backup automático de informações.</li>
</ol>
<p><strong>Requisitos Não Funcionais</strong></p>
<ol>
<li>Linguagem de programação
Desenvolver em Java ou C++.</li>
<li>Banco de Dados
Utilizar MySql.</li>
<li>Compatibilidade
Compatível a partir do Windows XP.</li>
<li>Tempo de consulta
O retorno das consultas feitas pelo usuário não deve ser maior que 15s</li>
<li>Usabilidade
Acesso fácil para usuários limitados.</li>
</ol>
