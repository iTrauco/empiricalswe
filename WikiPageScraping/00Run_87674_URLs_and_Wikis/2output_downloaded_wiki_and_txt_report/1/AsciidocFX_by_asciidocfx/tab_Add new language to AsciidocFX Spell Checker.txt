<p>AsciidocFX uses Morfologik library for the spellchecker. It needs a .dict file which is generated from a hunspell dictionary format and a .info file for a language.</p>
</div>
<div>
<p>If you want to add new language to AsciidocFX, for example for portuguese language <a href="https://github.com/languagetool-org/languagetool/tree/0fea351e439f94b933cd004e6d87756688101ae5/languagetool-language-modules/pt/src/main/resources/org/languagetool/resource/pt">https://github.com/languagetool-org/languagetool/tree/0fea351e439f94b933cd004e6d87756688101ae5/languagetool-language-modules/pt/src/main/resources/org/languagetool/resource/pt</a></p>
</div>
<div>
<dl>
<dt>NOTE</dt>
<dd>
<p>languagetool is a style and grammar checker library which used by OpenOffice, LibreOffice, Mozilla and etc.</p>
</dd>
</dl>
</div>
<div>
<p>Download portuguese.dict and portuguese.info then move these files to $HOME/.AsciidocFX/spellcheck</p>
</div>
