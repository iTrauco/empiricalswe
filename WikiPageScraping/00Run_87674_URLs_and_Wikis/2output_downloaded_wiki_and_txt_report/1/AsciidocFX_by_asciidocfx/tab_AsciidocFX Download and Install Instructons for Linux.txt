<p>Install AsciidocFX on Linux</p>
<div>
<ul>
<li>
<p>Download the "AsciidocFX_Linux.tar.gz" file here: <a href="https://github.com/asciidocfx/AsciidocFX#install-on-linux">https://github.com/asciidocfx/AsciidocFX#install-on-linux</a></p>
<div>
<ul>
<li>
<p>Locate the AsciidocFX Download File by Scrolling down the page to a table chart that says, "Table 1. Supported Operating Systems and Builds") + (Now find the AsciidocFX_Linux.tar.gz - not the one that says NO JRE -)</p>
<div>
<div>
<pre>Or Control+F and type Linux.tar should push you to the *AsciiddocFX_Linux.tar.gz*</pre>
</div>
</div>
</li>
</ul>
</div>
</li>
<li>
<p>Extract your tar file and cd into the main folder of AsciidocFX from terminal.
                                                             (remember to capitalize first letter  A and FX)
cd into folder AsciidocFX located in your downloads folder.</p>
</li>
<li>
<p>now type: sh ./AsciidocFX</p>
</li>
</ul>

</li>
