          <p>The <a href="https://github.com/DePierre/owtf_testing"><code>owtf_testing</code></a> repository aims to gather the test suites for the <a href="https://github.com/owtf/owtf"><code>OWTF</code> project</a>.</p>
<p>It has been developed during the <a href="https://www.owasp.org/index.php/Winter_Code_Sprint#OWASP_OWTF_-_Testing_Framework_Improvements_.28TAKEN.21.29" rel="nofollow">OWASP Winter Code Sprint 2014</a> that is now over.</p>
<p>In this Wiki, you will find all the information you need in order to install, configure and run the test suites. Plus, it also provides basic usages as well as explains how to interpret the results.</p>
