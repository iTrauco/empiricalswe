          <p>Welcome to the moodle-mod_exam wiki!</p>
<p>This is an activity module for Moodle. It allows the teacher to build quizzes with <strong>multiple choice question</strong> based on Quiz module. Main purpose of Exam module is to improve performance of quiz and to reduce database overhead, so that a large number of users can attempt quiz simultaneously.</p>
<p>Built of people who want to conduct large scale MCQ Quizzes.</p>
<p><strong>Features</strong></p>
<ul>
<li>High performance and less database overhead</li>
<li>No need to create quiz again, use already created quiz.</li>
<li>Simple and essay</li>
</ul>
<p><strong>Performance</strong></p>
<ul>
<li>Reduce a large number of database query, especially write quires to database.</li>
<li>Use of Moodle MUC</li>
</ul>
<p><strong>Limitation</strong></p>
<ul>
<li>Only multiple choice question supported</li>
<li>Allow only defferedfeedback questions behaviour.</li>
</ul>
