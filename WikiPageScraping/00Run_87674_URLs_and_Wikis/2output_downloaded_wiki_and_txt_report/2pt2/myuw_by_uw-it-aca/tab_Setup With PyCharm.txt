<a href="https://www.jetbrains.com/pycharm/download/" rel="nofollow">Download</a> and install PyCharm.</li>
<li>Open the MyUW virtualenv/project root folder.</li>
<li>Set the Python interpreter to your virtualenv: File -&gt; Settings -&gt; Project:myuw -&gt; Project Interpreter</li>
</ol>
<p>Click the Settings/Gear box and select "Add Local", and select the python file in your myuw/bin folder.</p>
<p>You should be all good to go! Try Ctrl-clicking an object to view its source.</p>
