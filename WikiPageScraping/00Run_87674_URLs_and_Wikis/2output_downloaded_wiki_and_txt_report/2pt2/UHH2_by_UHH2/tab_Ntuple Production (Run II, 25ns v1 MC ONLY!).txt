<p>Please sign up in the table below if you are processing a Run II sample. Enter the full sample name from DAS into the field “Sample name”. If you have processed a sample, you should upload an <code>xml</code> file to <code>UHH2/common/datasets/RunII_25ns_v1</code> with a list of files belonging to this dataset. Please choose a meaningful name for the <code>xml</code> file and enter it in the table (“Short name”). Cross sections for most of these processes can be found in <a href="https://twiki.cern.ch/twiki/bin/view/CMS/StandardModelCrossSectionsat13TeVInclusive" rel="nofollow">SM</a> and <a href="https://twiki.cern.ch/twiki/bin/viewauth/CMS/B2GMonteCarlo" rel="nofollow">B2G Signal</a>.</p>
