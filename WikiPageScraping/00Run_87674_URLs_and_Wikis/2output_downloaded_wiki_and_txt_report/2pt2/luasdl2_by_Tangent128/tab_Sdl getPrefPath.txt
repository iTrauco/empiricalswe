<p><strong>Available since <a href="Version-2-0-3-2">2.0.3-2</a></strong></p>
<p>Use this function to get the "pref dir". This is meant to be where the application can write personal files (Preferences and save games, etc.) that are specific to the application. This directory is unique per user and per application.</p>
