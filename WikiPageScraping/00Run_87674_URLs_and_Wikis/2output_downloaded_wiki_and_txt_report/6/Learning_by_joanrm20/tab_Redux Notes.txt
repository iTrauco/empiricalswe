<hr/>
<p>Represent whole seed as a single JavaScript Object
All mutations</p>
<ul>
<li>First principle of redux:
<ul>
<li>Everything that changes in your application including the data and UI state is contained in a single object called state or state tree.</li>

</li>

<ul>
<li>Second principle of redux:
<ul>
<li>The state is read-only, the only way to change the state tree is by dispatching an action. An action is a plain JS object describing in a minimum way what change in the app, whether it is initiated by a network request or by an user interaction.</li>
</ul>
</li>
</ul>
<h2>
<a aria-hidden="true" class="anchor" href="#pure-and-impure-functions" id="user-content-pure-and-impure-functions"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Pure and Impure Functions</h2>
<ul>
<li>
<p>Pure: Depends only on the values of their arguments; Do not have any observables side effects side effects like network or DB call, it calculates the new value. Do not modify the values passed to them.</p>
</li>
<li>
<p>Impure: They may have side effects, they may override the values it's passed to them. Some functions on Redux have to be pure.</p>
</li>
</ul>

<ul>
<li>Third principle of Redux:
<ul>
<li>The states mutations in the app need to be described as a pure function that takes the previous state and the action being dispatch and returns the next state of the app. It has to be a pure function. This function it's called the reducer.</li>
</ul>
</li>
</ul>

<pre><code>const createStore = (reducer) =&gt; {
  let state;
  let listeners = [];
  
  const getState = () =&gt; state;
  const dispatch = (action) =&gt; {
    state = reducer(state, action);
    listeners.forEach(listener =&gt; listener());
  };
  const subscribe = (listener) =&gt; {
    listeners.push(listener);
    return () =&gt; {
      listeners = listeners.filter(l =&gt; l !== listener);
    };
  };
  
  dispatch({});

  return { getState, dispatch, subscribe };
}

</code></pre>
