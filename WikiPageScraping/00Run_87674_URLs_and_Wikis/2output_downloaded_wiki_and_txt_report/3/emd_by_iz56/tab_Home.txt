<p>Installation notes - Create file 'emd.tmp' on path: C:\TMP\EMD.TMP;</p>
<p>Start the program by typing:</p>
<p>'EMD filename'</p>
<p>will read the file 'filename' into
EMD and display that file in the edit window.  Typing 'EMD' without a filename
will create an message error.</p>
<ul>
<li>Shift and Down  select&amp;copy text to Buffer;</li>
<li>/f  not display startup screen;</li>
</ul>
