<p>Welcome to the kalastack wiki!</p>
<p>Kalastack is a basic LEMP stack built for Drupal.
At its core it is a series of puppet manifests that are managed by Vagrant.
Kalastack was built to run primarily on Ubuntu Server 12.04,
though it will welcome multiple architectures in the future.</p>
