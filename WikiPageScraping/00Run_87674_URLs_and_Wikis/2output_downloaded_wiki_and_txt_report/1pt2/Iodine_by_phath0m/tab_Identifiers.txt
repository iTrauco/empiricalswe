          <p>In Iodine an identifier is string of alphanumeric characters used to identify a structure in code. An identifier can contain any value EXCEPT anything classified as a <a class="internal absent" href="/phath0m/Iodine/wiki/Keyword">Keyword</a></p>
<pre lang="bnf"><code>&lt;identifier&gt; ::= &lt;ident-char&gt; { &lt;ident-char&gt; | &lt;digit&gt; }
&lt;ident-char&gt; ::= &lt;letter&gt; | "_"
&lt;letter&gt;     ::= "a" ... "z" | "A" ... "Z"
&lt;digit&gt;      ::= "0" ... "9"
</code></pre>
