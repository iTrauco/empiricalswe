          <p>The <a class="internal present" href="/jrossignol/ContractConfigurator/wiki/WaypointGenerator-Behaviour">WaypointGenerator behaviour</a> can be accessed by referencing it by name in an expression:</p>
<pre><code>CONTRACT_TYPE
{
    name = MyContractType
    
    title = There are @/MyWaypointGenerator.Waypoints().Count() waypoints.

    BEHAVIOUR
    {
        name = MyWaypointGenerator
        type = WaypointGenerator

        ...
    }
}
</code></pre>
