<p>Before you can start using your new Tsunami, you'll need to set up your computer and the Arduino IDE to work with it.</p>
<h2>
<a aria-hidden="true" class="anchor" href="#installing-and-configuring-the-arduino-ide" id="user-content-installing-and-configuring-the-arduino-ide"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installing and configuring the Arduino IDE</h2>
<h3>
<a aria-hidden="true" class="anchor" href="#installing-arduino" id="user-content-installing-arduino"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installing Arduino</h3>
<p>If you do not already have the Arduino IDE installed, you will need to install it. If you already have it, check it is up to date: the Tsunami requires at least IDE 1.6 to operate. You can download the Arduino IDE <a href="http://www.arduino.cc/en/Main/Software" rel="nofollow">here</a>.</p>
<h3>
<a aria-hidden="true" class="anchor" href="#installing-the-tsunami-hardware-definitions" id="user-content-installing-the-tsunami-hardware-definitions"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installing the Tsunami hardware definitions</h3>
<p>Once you have installed the Arduino IDE, you need to install the Tsunami hardware definitions. First, open the Arduino preferences dialog:</p>
<p><img alt="" src="https://raw.githubusercontent.com/arachnidlabs/arachnidlabs-boards/docs/arduino-prefs.png"/></p>
<p>Down the bottom is a text field labelled "Additional Boards Manager URLs". Into that field, paste the following URL: <a href="https://raw.githubusercontent.com/arachnidlabs/arachnidlabs-boards/master/package_arachnidlabs.com_boards_index.json" rel="nofollow">https://raw.githubusercontent.com/arachnidlabs/arachnidlabs-boards/master/package_arachnidlabs.com_boards_index.json</a> then click OK.</p>
<p>If there are already URLs in that field, separate the new URL from the existing ones with a comma.</p>
<p>Next, click on <code>Tools -&gt; Board -&gt; Boards Manager</code>. The Arduino IDE will download a list of board definitions. Scrolling to the bottom, you should see a new entry for the Tsunami:</p>
<p><img alt="" src="https://raw.githubusercontent.com/arachnidlabs/arachnidlabs-boards/docs/arduino-hw.png"/></p>
<p>Highlight that entry and click "Install". Once installation is complete, close the board manager dialog. Congratulations, you have installed the Tsunami hardware definitions!</p>
<h3>
<a aria-hidden="true" class="anchor" href="#installing-the-tsunami-libraries" id="user-content-installing-the-tsunami-libraries"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installing the Tsunami libraries</h3>
<p>Next, click on <code>Sketch -&gt; Include Library -&gt; Manage Libraries</code>. The library manager will load. Click on the search box and enter "Tsunami". The entry for the Tsunami libraries should show up; click on that and click "Install". Once done, you can close the library manager dialog.</p>
<p>If you are using OSX or Linux, you're done! You can proceed to the Getting Started tutorial. If you're using Windows, you still have a couple of steps to do before you can use your Tsunami - read on!</p>
<h2>
<a aria-hidden="true" class="anchor" href="#installing-windows-drivers" id="user-content-installing-windows-drivers"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installing Windows drivers</h2>
<p>Windows requires drivers in order to interact with the Tsunami. These were included in the hardware definitions you downloaded in the previous steps; you just need to tell Windows how to find them.</p>
<h3>
<a aria-hidden="true" class="anchor" href="#windows-xp-and-windows-7" id="user-content-windows-xp-and-windows-7"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Windows XP and Windows 7</h3>
<p>Plug your Tsunami in and wait for Windows to recognise it and start the driver installation process. If the installer does not launch automatically, open Device Manager (<code>Start -&gt; Control Panel -&gt; Hardware</code>) and find the entry for "Arachnid Labs Tsunami" under 'Other Devices'. Right click and choose 'Update driver'.</p>
<p>At the next screen, choose "Browse my computer for driver software" and click next.</p>
<p>Click the 'Browse' button, and navigate to your Arduino sketchbook folder. You will find the drivers in a directory under that, called packages/arachnidlabs/hardware/avr/1.0.0/driver. Select that folder and click OK.</p>
<p>Windows will warn you that the driver has not passed Windows Logo testing. Click OK to continue.</p>
<p>Windows will tell you you have finished installing the drivers. You can now continue to the Getting Started tutorial.</p>
<h3>
<a aria-hidden="true" class="anchor" href="#windows-8" id="user-content-windows-8"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Windows 8</h3>
<p>Microsoft has made it rather harder to install unsigned drivers in Windows 8, unfortunately, so doing so involves some jumping through hoops.</p>
<p>First, open the charm bar by moving your cursor to the right hand side of the screen, and choose Settings. Then, click on Power. Hold down the shift key and click on 'Update and restart'. A new fullscreen menu will appear.</p>
<p>Click on 'Troubleshoot', then 'Advanced options', then 'Startup settings', then 'Restart'.</p>
<p>On the next screen, press 7 on your keyboard to select the option "Disable driver signature enforcement".</p>
<p>Once the computer finishes restarting, plug in your Tsunami. Windows will pop up a dialog, and you can follow instructions as described in the Windows 7 section above in order to install the drivers. Once you are done, proceed to the Getting Started tutorial.</p>
