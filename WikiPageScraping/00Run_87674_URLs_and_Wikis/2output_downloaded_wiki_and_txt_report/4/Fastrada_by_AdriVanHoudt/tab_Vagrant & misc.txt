<h4>
<a aria-hidden="true" class="anchor" href="#installation-1" id="user-content-installation-1"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Installation</h4>
<ol>
<li>Clone this repository using <code>git clone https://github.com/AdriVanHoudt/Fastrada.git</code>
</li>
<li>Open a terminal and navigate to the directory</li>
<li>
<strong>(Windows only)</strong>
<ol>
<li>If you're on Windows, run <code>dos2unix install.sh</code> and <code>dos2unix chef.rb</code>
</li>
<li>Go to the VagrantFile and comment out the line that has <code>:nfs =&gt; true</code> on the end and uncomment the second line that doesn't have that.</li>
</ol>
</li>
<li>Enter <code>vagrant plugin install vagrant-vbguest</code> (Upgrade guest additions)</li>
<li>Enter <code>vagrant up &lt;sitename&gt;</code> (example: <code>vagrant up teamb.localhost.com</code> for the development box)</li>
<li>Wait till it is completely done (Go grab some food or a drink :D)</li>
<li>When it is done press <code>vagrant ssh &lt;sitename&gt;</code> (example: <code>vagrant ssh teamb.localhost.com</code> for the vagrant box) to enter the terminal</li>
<li>Map <code>192.168.0.6</code> to <code>teamb.localhost.com</code> (Windows: C:\Windows\System32\drivers\etc\hosts, Ubuntu, mac: /etc/hosts)</li>
<li>Have fun using Vagrant :D</li>
</ol>
