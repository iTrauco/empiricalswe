<h4>
<a aria-hidden="true" class="anchor" href="#what-is-4jahn" id="user-content-what-is-4jahn"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>What is 4jahn?</h4>
<blockquote>
<p>4jhan is a minimal image board server. Contrary to conventional image boards, <a href="http://www.4chan.org/" rel="nofollow">4chan</a> or <a href="http://www.2chan.net/" rel="nofollow">2chan</a> only hosts the data without a UI. This makes it easier for anyone to run a server with port-forwarding on a device like the <a href="https://www.raspberrypi.org/" rel="nofollow">Raspberry Pi</a>. It's basically just a API server.</p>
</blockquote>
<h4>
<a aria-hidden="true" class="anchor" href="#how-to-use-it" id="user-content-how-to-use-it"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>How to use it?</h4>
<blockquote>
<p>If you don't enjoy reading JSON and sending manual POST requests, you're going to need a client to interpret the data. We are still working on a client, and it should be finished <em>in the future</em>. Currently this <a href="https://gist.github.com/phikal/3254a5cd996c27527431">Minimal HTTP Client</a> exists, for creating posts and commenting.</p>
</blockquote>
