<p><a href="https://help.ubuntu.com/12.04/serverguide/openssh-server.html" rel="nofollow">Tutorial</a><br/>
<a href="https://www.youtube.com/watch?v=AyxPb-74r4U" rel="nofollow">Video Tutorial</a></p>
<p>Remote Host (ex. 2.2.2.2)</p>
<pre><code>$ sudo useradd -m web
$ sudo usermod web -s /bin/bash
$ sudo passwd web

// Authenticated user should have read and write permissions
$ chmod 600 /home/web/.ssh/authorized_keys    
</code></pre>
<p>Client</p>
<pre><code>$ ssh-keygen -t rsa
$ ssh-copy-id web@2.2.2.2
</code></pre>
<p>Another way of doing it.</p>
<p>Client</p>
<pre><code>$ scp id_rsa.pub web@2.2.2.2:/home/web/.ssh/id_rsa.pub 
</code></pre>
<p>Remote host</p>
<pre><code>$ cd /home/web/.ssh
$ cat id_rsa.pub &gt;&gt; authorized_keys
</code></pre>
