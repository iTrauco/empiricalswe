          <h2>
<a aria-hidden="true" class="anchor" href="#json-http-post-notifications" id="user-content-json-http-post-notifications"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>JSON HTTP POST Notifications</h2>
<ul>
<li>
<strong>Source</strong>: n/a</li>
<li>
<strong>Icon Support</strong>: No</li>
<li>
<strong>Message Format</strong>: JSON</li>
<li>
<strong>Message Limit</strong>: 32768 Characters per message</li>
</ul>
<p>This is just a custom Notification that allows you to have this tool post to a web server as a simple JSON string. This is useful for those who want to be notified via their own custom methods.</p>
<p>The format might look something like this:</p>
<div class="highlight highlight-source-json"><pre>{
   <span class="pl-s"><span class="pl-pds">"</span>version<span class="pl-pds">"</span></span>: <span class="pl-s"><span class="pl-pds">"</span>1.0<span class="pl-pds">"</span></span>,
   <span class="pl-s"><span class="pl-pds">"</span>title<span class="pl-pds">"</span></span>: <span class="pl-s"><span class="pl-pds">"</span>What A Great Movie Downloaded Successfully<span class="pl-pds">"</span></span>,
   <span class="pl-s"><span class="pl-pds">"</span>message<span class="pl-pds">"</span></span>: <span class="pl-s"><span class="pl-pds">"</span>Plenty of details here<span class="pl-pds">"</span></span>,
   <span class="pl-s"><span class="pl-pds">"</span>type<span class="pl-pds">"</span></span>: <span class="pl-s"><span class="pl-pds">"</span>info<span class="pl-pds">"</span></span>
}</pre></div>
<p>The <em>type</em> will be one of the following:</p>
<ul>
<li>
<strong>info</strong>: An informative type message</li>
<li>
<strong>success</strong>: A successful report</li>
<li>
<strong>failure</strong>: A failure report</li>
<li>
<strong>warning</strong>: A warning report</li>
</ul>
<h3>
<a aria-hidden="true" class="anchor" href="#syntax" id="user-content-syntax"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a>Syntax</h3>
<p>Valid syntaxes are as follows:</p>
<ul>
<li>
<strong>json</strong>:://<strong>{hostname}</strong>
</li>
<li>
<strong>json</strong>:://<strong>{hostname}</strong>:<strong>{port}</strong>
</li>
<li>
<strong>jsons</strong>:://<strong>{hostname}</strong>
</li>
<li>
<strong>jsons</strong>:://<strong>{hostname}</strong>:<strong>{port}</strong>
</li>
</ul>
