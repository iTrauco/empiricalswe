        <div class="markdown-body">
          <p>My first experience with the <a href="https://github.com/Reactive-Extensions/RxJS/blob/master/examples/canvaspaint/canvaspaint.js">Canvas Paint</a> example ran into issues due to an external distraction while I was reading and entering the code before debugging.</p>
<p>I had not entered closing lines:</p>
<pre><code>    main();
}(window));
</code></pre>
<p>which led to nothing launching any activity. Took a while of first suspecting some browser specific or packaging issue before discovering this goofy omission.</p>
<p>After that I had fun tweaking the skip setting in the mouseDiffs function:</p>
<pre><code>            var mouseDiffs = mouseMoves.zip(mouseMoves.skip(10), function(fst, snd) {
                return {first: getOffset(fst), second: getOffset(snd) };
            });
</code></pre>
<p>A skip of ten produced interesting effects as it would lead to the strokes skipping 10 coordinates each, creating a sort of <a href="http://en.wikipedia.org/wiki/Spirograph" rel="nofollow">Spirograph</a> effect.</p>
