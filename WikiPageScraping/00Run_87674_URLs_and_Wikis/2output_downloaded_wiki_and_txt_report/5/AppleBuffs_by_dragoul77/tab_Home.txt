          <p>Welcome to the AppleBuffs wiki!</p>
<p>AppleBuffs is a simple minecraft forge mod, which adds new items (apples) with special buffs.
The mod is inspired by the Tinkers' Construct <em>Jeweled Apple</em> (it also uses the same logic).</p>

<p>The mod brings it's own config file <code>AppleBuffs.cfg</code>. It will be created when fist launched. The configuration looks like this:</p>
<pre><code>buffmultipliers {  
    I:diamondApple=1  
    I:emeraldApple=2  
    I:emerondApple=3  
    I:godDiamondApple=9  
    I:godEmeraldApple=18  
    I:godEmerondApple=27  
}  
  
useapples {  
    B:diamondApple=true  
    B:emeraldApple=true  
    B:emerondApple=true  
    B:godDiamondApple=true  
    B:godEmeraldApple=true  
    B:godEmerondApple=true  
}  
</code></pre>
<p>All items can be enabled/disabled in the configuration. Also the duration of the buffs can be configured through the config.</p>
