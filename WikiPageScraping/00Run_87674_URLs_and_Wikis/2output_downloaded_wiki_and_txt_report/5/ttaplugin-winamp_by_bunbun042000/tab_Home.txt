          <p>"ttaplugin-winamp" is input plugin for winamp to play <a href="http://en.true-audio.com/" rel="nofollow">The True Audio</a> file.</p>
<p>This source is based on <a href="http://en.true-audio.com/Free_Downloads" rel="nofollow">ttaplugin-winamp</a>.</p>
<p>Replace decode engine to <a href="http://sourceforge.net/projects/tta/" rel="nofollow">libtta++</a>.
Add Winamp media library extension (including album art reading) using <a href="https://github.com/bunbun042000/taglib-modified">my taglib-modified</a>.</p>
<p>If you need installer file, please download <a href="https://github.com/bunbun042000/ttaplugin-winamp/releases/download/Beta18/ttaplugin-winamp_v3_2_Beta18.zip">ttaplugin-winamp_v3_2_Beta18.zip</a>.</p>
