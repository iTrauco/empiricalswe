          <p>Welcome to the primestationone wiki!</p>
<p>Your best bet is probably going to be the Wiki pages for the emulators and ports used from RetroPie, which can be found at the following location:
<a href="https://github.com/petrockblog/RetroPie-Setup/wiki">https://github.com/petrockblog/RetroPie-Setup/wiki</a></p>
<p>Additionally to enabling netplay and all the emulators, stable as well as experimental (such as Sega Dreamcast and Sega Saturn) found in RetroPie, the PrimeStation One adds the following to EmulationStation:</p>
<p>A custom PrimeStation One theme</p>
<p>A "Settings" page with useful configuration and installation scripts (the first PS3 controller icon)</p>
<p>Additional Ports:</p>
<ul>
<li>Angband</li>
<li>Descent 1 shareware</li>
<li>Descent 2 shareware</li>
<li>LXDE XWindow manager / startx (Windowed mode linux)</li>
</ul>
<p>Wifi adapter support with connection manager (accessible via "Settings" page in EmulationStation)</p>
